预处理器 在编译器看到代码之前, C 源文件首先通过宏预处理器, 即 CPP. CPP 命令以 “#” 开头: 插入 file.h 文件 1#include “file.h” 在标准位置查找文件 1#include <stdio.h> 定义常量 1#define M_PI (3.14) 有条件地包含文件 1#if/#endif 预处理中定义 (#define) 的"函数"不是真正的函数, 它只是更改程序的文本; 即使是 #include, 也只是将文件内容拷贝到当前文件中. 例如: 1#define twox(x) (x + x); 只是用 (x + x) 替换 twox(x), 所以调用 twox(3), 相当于调用 (3 + 3). 预处理器还会用一个空格替换掉注释, 这样 C 编译器就不需要费力跳过注释.
