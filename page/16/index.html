Git д»Ӣз»Қ Git жҳҜдёҖдёӘеҲҶеёғејҸзүҲжң¬жҺ§еҲ¶зі»з»ҹ, иҖҢдёҚжҳҜйӣҶдёӯејҸзҡ„зүҲжң¬жҺ§еҲ¶зі»з»ҹ, еҚіжҜҸдёӘејҖеҸ‘дәәе‘ҳзҡ„з”өи„‘йғҪеӯҳеӮЁж•ҙдёӘйЎ№зӣ®зҡ„еҺҶеҸІ (жүҖжңүжҸҗдәӨи®°еҪ•), жҲ‘д»¬е°Ҷж•ҙдёӘйЎ№зӣ®зҡ„еҺҶеҸІз§°дёә вҖңеӯҳеӮЁеә“вҖқ. Git зҡ„дё»иҰҒз»„жҲҗйғЁеҲҶеҰӮеӣҫжүҖзӨә: еҜ№иұЎ Git дёӯзҡ„еҜ№иұЎеҰӮдёӢжүҖзӨә: Blob еҜ№иұЎ: еңЁ git дёӯеӯҳеӮЁж–Үд»¶еҶ…е®№зҡ„еҜ№иұЎ. Blob зӣёжҜ”жҷ®йҖҡж–Үд»¶дёҚеҢ…еҗ«е…ғж•°жҚ® (дҫӢеҰӮ: ж–Үд»¶еҲӣе»әж—¶й—ҙзӯү), Blob еҸӘжңүеҶ…е®№ (ж•°жҚ®зҡ„дәҢиҝӣеҲ¶жөҒ). git дёӯзҡ„ Blob йҖҡиҝҮ SHA-1 е“ҲеёҢеҖје”ҜдёҖж ҮиҜҶ (еҜ№дәҺзӣёеҗҢеҶ…е®№зҡ„ Blob, еҚідҪҝеңЁдёӨеҸ°дёҚеҗҢзҡ„з”өи„‘дёҠ, дә§з”ҹзҡ„ SHA-1 е“ҲеёҢеҖјд»Қз„¶зӣёеҗҢ). Tree еҜ№иұЎ: дёҖдёӘзӣ®еҪ•еҲ—иЎЁ, еј•з”Ёе…¶д»– Blob е’Ң Tree еҜ№иұЎ, е№¶еӯҳеӮЁе…¶д»–еҜ№иұЎеҜ№еә”зҡ„еҗҚеӯ—. з”Ё SHA-1 е“ҲеёҢеҖје”ҜдёҖж ҮиҜҶ, йҖҡиҝҮе…¶д»–еҜ№иұЎзҡ„ SHA-1 еј•з”Ёе®ғд»¬.
