集合论

集合

定义

定义
定义明确的对象的集合. 对象被称为元素或集合的成员.

表示
集合通常用大写字母表示, 可以通过列出元素和用大括号包围元素来描述或定义.

  • 例如: A = {2, 3, 5, 7, 11} 或 A = {ab\frac{a}{b}ba​ | a, b 为整数, b ≠\neq= 0}.

如果 x 是 A 的一个元素, 则写作 x ∈\in∈ A, 如果 y 不是 A 的元素, 则写作 y ∉\notin∈/ A.

如果两个集合 A 和 B 拥有相同的元素(元素的顺序和重复不重要), 则称集合 A 等于集合 B , 写作 A = B.

基数

定义
一个集合中元素的个数, 称为集合的基数 (势), 写作 |A|.

  • 例如: A = {2, 3, 5, 7}, A 的基数为4, 即|A| = 4.

一个集合的基数可以为0, 称为空集, 写作{} 或 ∅\emptyset∅ .一个集合也可以拥有无数个元素.

子集和真子集

子集
如果一个集合 A 的每个元素都在集合 B 中, 那么 A 是 B 的一个子集, 写作 A ⊆\subseteq⊆ B, B 是 A 的超集, 写作 B ⊇\supseteq⊇ A.

  • 例如: A = {1, 2, 3}, B = {1, 2, 3, 4}, C = {1, 2, 3}, 则 A 是 B 的一个子集, A 也是 C 的一个子集.

真子集
如果一个集合 A 的每个元素都在集合 B 中, 并且 B 中至少有一个元素不在 A 中, 则称 A 为 B 的一个真子集, 写作A ⊂\subset⊂ B.

  • 例如: A = {1, 2, 3}, B = {1, 2, 3, 4}, C = {1, 2, 3}, 则 A 是 B 的一个真子集, 但A 不是 C 的一个真子集.

空集是任何非空集合的真子集, 是任何集合的子集, 每个集合都是自身的子集.

交集和并集

交集
包含集合 A 和 集合 B 中共有的元素的集合, 称为集合 A 和集合 B 的交集, 写作 A ∩\cap∩ B, 如果A ∩\cap∩ B = ∅\emptyset∅, 则称集合 A 和集合 B 不相交.

  • 例如: A = {1, 2, 3}, B = {2}, 则 A ∩\cap∩ B = {2}.

并集
包含集合 A 和 集合 B 中所有元素的集合, 称为集合 A 和集合 B 的并集, 写作 A ∪\cup∪ B.

  • 例如: A = {1, 2}, B = {5, 1}, 则 A ∪\cup∪ B = {1, 2, 5}.

性质

  • A ∪\cup∪ B = B ∪\cup∪ A
  • A ∪\cup∪ ∅\emptyset∅ = A
  • A ∩\cap∩ B = B ∩\cap∩ A
  • A ∩\cap∩ ∅\emptyset∅ = ∅\emptyset∅

补集

如果 A 和 B 是两个集合, 那么 A 在 B 中的相对补集, 或者 B 和 A 之间的集合差, 是一个包含 B 中元素, 但不包含 A 中元素的集合, 写作 B - A 或 B \ A, B \ A = {x ∈\in∈ B | x ∉\notin∈/ A}.

  • 例如: A = {1, 2, 3}, B = {3 ,5, 9}, B \ A = {5, 9}.

性质

  • A \ A = ∅\emptyset∅
  • A \ ∅\emptyset∅ = A
  • ∅\emptyset∅ \ A = ∅\emptyset∅

常见集合

  • N\mathbb{N}N 表示所有自然数的集合: {0, 1, 2, 3, …}.

  • Z\mathbb{Z}Z 表示所有整数的集合: {… , -2, -1, 0, 1, 2, …}.

  • Q\mathbb{Q}Q 表示所有实数的集合: {ab\frac{a}{b}ba​ | a, b ∈\in∈ Z, b ≠\neq= 0}.

  • R\mathbb{R}R 表示所有有理数的集合.

  • C\mathbb{C}C 表示所有复数的集合.

笛卡尔积和幂集

笛卡尔积
两个集合 A 和 B 的笛卡尔积(也叫交叉乘积), 是所有第一个部分是 A 的元素, 第二个部分是 B 的元素的对的集合, 写作 A x B, A x B = {(a, b) | a ∈\in∈ A, b ∈\in∈ B}.

  • 例如: A = {1, 2, 3}, B = {u, v}, A x B = {(1, u), (1, v), (2, u), (2, v), (3, u), (3, v)}.

幂集
给定一个集合 S , S 的幂集是 S 所有子集的集合, 写作 P(S), P(S) = {T | T ⊆\subseteq⊆ S}.

  • 例如: S = {1, 2, 3}, P(S) = { {1}, {2}, {3}, {1, 2}, {1, 3}, {2, 3}, {1, 2, 3} }.

如果|S| = k, 则 |P(S)| = 2k2^{k}2k.
