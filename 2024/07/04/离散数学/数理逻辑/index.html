命题逻辑

命题

  • 命题是一个简单的陈述, 不能包含模糊的术语, 要么是真, 要么是假.

    例如:
    3\sqrt{3}3​ 是无理数 是
    1 + 1 = 5 是
    我每天早饭吃2个鸡蛋 是
    2 + 2 不是
    x2x^{2}x2 + 3x3x3x = 5 不是 (xxx 为未知数)

  • 命题形式: 带有变量的语句.

真值表

描述命题形式的可能值, 列出所有变量的可能输入值, 然后列出对应的输出值 (顺序不重要).

逻辑

合取
P ∧\wedge∧ Q, 当命题 P 和命题 Q 都为真时结果才为真.

析取
P ∨\vee∨ Q, 只要命题 P 和命题 Q中有一个为真, 结果就为真.

否定
¬\neg¬ P, 当命题 P 为假时结果为真.

蕴涵
P ⇒\Rightarrow⇒ Q, 当命题 P 为真, 命题 Q 为假时, 结果才为假.

有许多不同的表达形式:
1、如果 P 则 Q.
2、Q 如果 (if) P.
3、P 只有 (only if) Q.
4、P 是 Q 的充分条件.
5、Q 是 P 的必要条件.
6、Q 除非不是 P.

P ⇒\Rightarrow⇒ Q 的逆反命题 ¬\neg¬ Q ⇒\Rightarrow⇒ ¬\neg¬ P 和 P ⇒\Rightarrow⇒ Q 拥有相同的真值, 所以它们在逻辑上是等价的.

等价
如果 P ⇒\Rightarrow⇒ Q 和 Q ⇒\Rightarrow⇒ P 都为真, 即 P 和 Q 拥有相同的真值, 则称 P 和 Q 等价, 也称"P if and only if Q"或"P iff Q", 写作 P ⇐⇒\Leftarrow\Rightarrow⇐⇒ Q.

排中律

对于任意命题 P, 要么 P 为真, 要么 ¬\neg¬ P 为真, 因此 P ∨\vee∨ ¬\neg¬ P 一定为真.

重言式

一个命题形式, 无论其变量的真值如何, 结果都为真.

矛盾式

一个命题形式, 无论其变量的真值如何, 结果都为假.

德摩根定律

  • ¬\neg¬ (P ∧\wedge∧ Q) ≡\equiv≡ (¬\neg¬ P ∨\vee∨ ¬\neg¬ Q)

  • ¬\neg¬ (P ∨\vee∨ Q) ≡\equiv≡ (¬\neg¬ P ∧\wedge∧ ¬\neg¬ Q)

  • ¬\neg¬ (∀\forall∀ x P(x)) ≡\equiv≡ ∃\exists∃ x ¬\neg¬ P(x)

  • ¬\neg¬ (∃\exists∃ x P(x)) ≡\equiv≡ ∀\forall∀ x ¬\neg¬ P(x)

量词

普遍性

普遍性量词表示"对于所有", 写作 ∀\forall∀ .

例如: "对于所有自然数 n, n2n^{2}n2 + n + 41 是素数"可以写作 (∀\forall∀ n ∈\in∈ N\mathbb{N}N) (n2n^{2}n2 + n + 41 是素数).

存在性

存在性量词表示"存在", 写作 ∃\exists∃ .

例如: “有一个整数小于2, 但它的平方大于4” 可以写作 (∃\exists∃ x ∈\in∈ Z\mathbb{Z}Z) (n < 2 并且 n2n^{2}n2 > 4).

宇宙

定义
变量所属范围, 包含变量可能值的集合.

例如: “有一些哺乳动物会下蛋”, 可以表述成"存在一种会下蛋的哺乳动物 x", 用大写字母 U\mathbb{U}U 代表哺乳动物, 用数学语言来表示就是 (∃\exists∃ x ∈\in∈ U\mathbb{U}U) (x 会下蛋), 这里的 U\mathbb{U}U 便是哺乳动物, 将 x 的范围限制在哺乳动物之中, 如果 U\mathbb{U}U 是定义明确的, 甚至可以省略它, 写作 ∃\exists∃ x (x 会下蛋).

性质
如果宇宙是有限的, 则可以通过析取和合取来表示命题而无需使用量词.

例如: U\mathbb{U}U = {1, 2, 3}, (∃\exists∃ x ∈\in∈ U\mathbb{U}U) P(x) ⇐⇒\Leftarrow\Rightarrow⇐⇒ P(1) ∨\vee∨ P(2) ∨\vee∨ P(3), (∀\forall∀ x ∈\in∈ U\mathbb{U}U) P(x) ⇐⇒\Leftarrow\Rightarrow⇐⇒ P(1) ∧\wedge∧ P(2) ∧\wedge∧ P(3).

辖域

  • 若量词后有括号, 则括号内的子公式就是量词的辖域.

    (∀\forall∀ x)(…)

  • 若量词后没有括号, 则与量词邻接的子公式就是该量词的辖域.

    (∀\forall∀ x) F(x)

  • (∃\exists∃ x) P(x) ∨\vee∨ (∃\exists∃ x) Q(x) P(x) 中的 x 和 Q(x) 中的 x 不是同一个 x, 因为两者不在同一个辖域中.

  • (∀\forall∀ x ∈\in∈ Z\mathbb{Z}Z) (∃\exists∃ y ∈\in∈ Z\mathbb{Z}Z) (y > x): 给定一个整数, 可以找到一个比它更大的数.

  • (∃\exists∃ y ∈\in∈ Z\mathbb{Z}Z) (∀\forall∀ x ∈\in∈ Z\mathbb{Z}Z) (y > x): 整数拥有最大值.

自由变元

若变元 x 不出现在使用变元的量词的辖域之内, 即 x 的出现不是约束出现, 则称 x 的出现为自由出现, x 为自由变元.

例如: (∃\exists∃ x) P(x) ∨\vee∨ Q(x, y)

  • P(x) 中的 x 为约束变元, Q(x, y) 中的 x, y 为自由变元.

约束变元

若变元 x 出现在使用变元的量词的辖域之内, 则称 x 的出现为约束出现, 此时的变元 x 为约束变元.

形式

  • 要证 P ⇒\Rightarrow⇒ Q, 如果 ¬Q⇒¬P\neg Q \Rightarrow \neg P¬Q⇒¬P 更容易证明, 则可以通过证明 ¬Q⇒¬P\neg Q \Rightarrow \neg P¬Q⇒¬P 来证明 P ⇒\Rightarrow⇒ Q, 因为一个命题在蕴涵逻辑上和它的逆反命题等价.

形式
要证明 P 是正确的, 可以先假设 ¬\neg¬ P 是正确的, 然后通过 ¬\neg¬ P 推出来的结果与事实矛盾或推出两个完全矛盾的结果, 则证明 P 是正确的.

目的: 证明 P

解: 假设 ¬\neg¬ P ⇒\Rightarrow⇒ Q … ⇒\Rightarrow⇒ … ¬\neg¬ Q, 则推出 ¬\neg¬ P ⇒\Rightarrow⇒ ¬\neg¬ Q ∧\wedge∧ Q, 则 P 为真.

弱归纳法

数学归纳法

形式

  • 假设要证明 ∀n∈N\forall n \in \mathbb{N}∀n∈N, P(n) 成立, 证明这一点需要三个步骤:
    • 基本情况: 证明 P(a) 成立, a 不一定要为 0, 根据问题的情况而定.
    • 归纳假设: 对于 ∀k≥0\forall k \geq 0∀k≥0 , 假设P(k) 都成立.
    • 归纳步骤: 根据归纳假设, 证明 P(k + 1) 也成立.
  • 可以将这三个步骤想象成多米诺骨牌, 其中 P(i) 表示第 i 块骨牌, 将这些骨牌排成一列, 碰倒第 i 块骨牌相当于证明 P(i) 成立, 归纳假设和归纳步骤表明, 如果第 k 块骨牌被碰倒, 则第 k + 1 块骨牌也会被碰倒, 基本情况就是碰倒这一列骨牌中的第一块, 显然当一列中的第一块骨牌被碰倒时, 会发生连锁反应, 最后所有的骨牌都将被碰倒.

    ∵(k+1)3−(k+1)=k3+3k2+3k+1−(k+1)\because (k + 1)^3 - (k + 1) = k^3 + 3k^2 + 3k + 1 - (k + 1)∵(k+1)3−(k+1)=k3+3k2+3k+1−(k+1)

    =(k3−k)+3k2+3k= (k^3 - k) + 3k^2 + 3k=(k3−k)+3k2+3k

例二

  • 证明: (两色定理) "地图"是一个矩形, 可以通过绘制直线将地图划分成多个区域, 每条直线可以将矩形划分成两个不同的区域, 使用不超过两种颜色为地图着色, 使地图上的任意两个区域都具有不同的颜色.

    我们可以保持这条线一侧的颜色不变, 将这条线另一侧的颜色颠倒.

    例如: 假设已经使用了红色和蓝色对图进行染色, 我们可以保持第 k + 1条线左侧的颜色不变, 右侧所有区域的颜色从原来的颜色变为另一个颜色, 即 红 ⇐⇒\Leftarrow \Rightarrow⇐⇒ 蓝, 如图:

    此时第 k + 1 条线两端的区域具有不同的颜色, 根据归纳假设, 前 k 条线两端的区域都具有不同的颜色, 则 P (k + 1) 成立.

加强归纳假设

形式

  • 当归纳假设太弱, 无法提供足够的结构来证明 k + 1 的情况时, 使用加强归纳假设.
  • 通过观察前 k 项寻找更强的归纳假设, 即能够提供更多的结构来证明.

例一

  • 证明: **∀n≥1\forall n \geq 1∀n≥1, ∑i=1n1i2≤2\sum_{i = 1}^{n} \frac{1}{i^2} \leq 2∑i=1n​i21​≤2 . **

  • 目的: (∀n∈Z\forall n \in \mathbb{Z}∀n∈Z) (n≥1n \geq 1n≥1) ⇒\Rightarrow⇒ ∑i=1n1i2≤2\sum_{i = 1}^{n} \frac{1}{i^2} \leq 2∑i=1n​i21​≤2 .

  • 解:

    首先尝试直接使用数学归纳法:

    • 基本情况 (n = 1): 1i2=1≤2\frac{1}{i^2} = 1 \leq 2i21​=1≤2 , 则 P(1) 成立.
    • 归纳假设 (n = k): 假设对于 k≥1k \geq 1k≥1 , ∑i=1k1i2≤2\sum_{i = 1}^{k} \frac{1}{i^2} \leq 2∑i=1k​i21​≤2 .
    • 归纳步骤 (n = k + 1): ∑i=1k+11i2=∑i=1k1i2+1(k+1)2≤2+1(k+1)2\sum_{i = 1}^{k + 1} \frac{1}{i^2} = \sum_{i = 1}^{k} \frac{1}{i^2} + \frac{1}{(k + 1)^2} \leq 2 + \frac{1}{(k + 1)^2}∑i=1k+1​i21​=∑i=1k​i21​+(k+1)21​≤2+(k+1)21​ .

    此时归纳假设太弱, 无法提供足够的结构来证明 2+1(k+1)2≤22 + \frac{1}{(k + 1)^2} \leq 22+(k+1)21​≤2 .

    现在尝试加强归纳假设, 假设 ∀n≥1\forall n \geq 1∀n≥1, ∑i=1n1i2≤2−1n\sum_{i = 1}^{n} \frac{1}{i^2} \leq 2 - \frac{1}{n}∑i=1n​i21​≤2−n1​ .

    • 基本情况 (n = 1): 1i2=1≤2−11=1\frac{1}{i^2} = 1 \leq 2 - \frac{1}{1} = 1i21​=1≤2−11​=1 , 则 P(1) 成立.
    • 归纳假设 (n = k): 假设对于 k≥1k \geq 1k≥1 , ∑i=1k1i2≤2−1k\sum_{i = 1}^{k} \frac{1}{i^2} \leq 2 - \frac{1}{k}∑i=1k​i21​≤2−k1​ .
    • 归纳假设 (n = k + 1): ∑i=1k+11i2=∑i=1k1i2+1(k+1)2≤2−1k+1(k+1)2≤2−1k+1\sum_{i = 1}^{k + 1} \frac{1}{i^2} = \sum_{i = 1}^{k} \frac{1}{i^2} + \frac{1}{(k + 1)^2} \leq 2 - \frac{1}{k} + \frac{1}{(k + 1)^2} \leq 2 - \frac{1}{k + 1}∑i=1k+1​i21​=∑i=1k​i21​+(k+1)21​≤2−k1​+(k+1)21​≤2−k+11​ , 则 P(k + 1) 成立.

    根据数学归纳法的定义可得原命题成立.

强归纳法

形式

  • 强归纳类似于弱归纳, 不仅仅假设 P(k) 成立, 而是假设 P(0), P(1), P(2), …, P(k) 都成立. 强归纳和弱归纳能完成的东西是相同的, 即强归纳和弱归纳等价的, 但强归纳更容易使用.

  • 强归纳为什么和弱归纳等价?

    可以假设 Q(n)=P(0)∧P(1)∧...∧P(n)Q(n) = P(0) \wedge P(1) \wedge ... \wedge P(n)Q(n)=P(0)∧P(1)∧...∧P(n), 则对 P(n) 使用强归纳来证明 P(n) 为真相当于对 Q(n) 使用弱归纳来证明 Q(n)为真, P(n) 对所有 n 为真当且仅当 Q(n) 对所有 n 为证真, 即对 P(n) 使用强归纳可以转化为对 Q(n) 使用弱归纳, 同理, 对 Q(n) 使用弱归纳可以转化为对 P(n) 使用强归纳, 则弱归纳和强归纳等价.

  • 假设要证明 ∀n∈N\forall n \in \mathbb{N}∀n∈N, P(n) 成立, 证明这一点需要三个步骤:

    • 基本情况: 证明 P(a) 成立, a 为有限的情况, 如果有必要, 归纳假设的有效性需要多个基本情况的支持.
    • 归纳假设: 假设 ∀n,a≤n≤k\forall n, a \leq n \leq k∀n,a≤n≤k, P(n) 均成立.
    • 归纳步骤: 根据归纳假设, 证明 P(k + 1) 也成立.

例一

  • 证明: 对于所有大于等于12的自然数 n , n=4x+5yn = 4x + 5yn=4x+5y (x,y∈Nx, y \in \mathbb{N}x,y∈N).

  • 目的: (∀n∈N\forall n \in \mathbb{N}∀n∈N) (n≥12n \geq 12n≥12) ⇒\Rightarrow⇒ n=4x+5yn = 4x + 5yn=4x+5y (x,y∈Nx, y \in \mathbb{N}x,y∈N).

  • 解:

    • 基本情况 (n = 12): 12=4⋅3+5⋅012 = 4 \cdot 3 + 5 \cdot 012=4⋅3+5⋅0 , 即 P(12) 成立.
    • 基本情况 (n = 13): 13=4⋅2+5⋅113 = 4 \cdot 2 + 5 \cdot 113=4⋅2+5⋅1 , 即 P(13) 成立.
    • 基本情况 (n = 14): 14=4⋅1+5⋅214 = 4 \cdot 1 + 5 \cdot 214=4⋅1+5⋅2 , 即 P(14) 成立.
    • 基本情况 (n = 15): 15=4⋅0+5⋅315 = 4 \cdot 0 + 5 \cdot 315=4⋅0+5⋅3 , 即 P(15) 成立.
    • 归纳假设: 假设当 12≤n≤k12 \leq n \leq k12≤n≤k (k≥15k \geq 15k≥15) 时, P(n) 成立.
    • 归纳步骤:

    ∵k≥15⇒n=k+1≥16⇒(k+1)−4≥12\because k \geq 15 \Rightarrow n = k + 1 \geq 16 \Rightarrow (k + 1) - 4 \geq 12∵k≥15⇒n=k+1≥16⇒(k+1)−4≥12

    设 m=(k+1)−4m = (k + 1) - 4m=(k+1)−4 , 则 m≥12m \geq 12m≥12 , 根据归纳假设可得 P(m) 成立,

    即 m=(k+1)−4=4x′+5y′m = (k + 1) - 4 = 4x' + 5y'm=(k+1)−4=4x′+5y′ (x′,y′∈Nx', y' \in Nx′,y′∈N), 设 x=x′+1,y=y′x = x' + 1, y = y'x=x′+1,y=y′,

    则 (k+1)−4=4⋅(x−1)+5⋅y⇒n=k+1=4⋅x+5⋅y(k + 1) - 4 = 4 \cdot (x - 1) + 5 \cdot y \Rightarrow n = k + 1 = 4 \cdot x + 5 \cdot y(k+1)−4=4⋅(x−1)+5⋅y⇒n=k+1=4⋅x+5⋅y , 即 P(n) 成立.

    根据强归纳法的定义, 可得原命题成立.

例二

  • 证明: 每个大于1的自然数都可以写成至少一个素数的乘积.
  • 目的: (∀n∈N\forall n \in \mathbb{N}∀n∈N) (n > 1) ⇒\Rightarrow⇒ (n=p1⋅p2⋅...⋅pkn = p_1 \cdot p_2 \cdot ... \cdot p_kn=p1​⋅p2​⋅...⋅pk​) (p1,p2,...,pkp_1, p_2, ..., p_kp1​,p2​,...,pk​ 为素数).
  • 解:
    • 基本情况 (n = 2): 因为2是素数, 得 P(2) 成立.
    • 归纳假设: 假设当 2≤n≤k2 \leq n \leq k2≤n≤k 时, P(n) 成立.
    • 归纳步骤: 当 n = k + 1 时, 有两种情况
      • n = k + 1 为素数, 则 P(n) 显然成立.
      • n = k + 1 不为素数, 假设 n=x⋅yn = x \cdot yn=x⋅y (1<x,y<n=k+11 < x, y < n = k + 11<x,y<n=k+1 且 x,y∈Zx, y \in \mathbb{Z}x,y∈Z), 根据归纳假设, P(x), P(y) 成立, 即 x=px1⋅px2⋅px3...pxkx = p_{x1} \cdot p_{x2} \cdot p_{x3} ... p_{xk}x=px1​⋅px2​⋅px3​...pxk​ (px1,px2,px3,...,pxkp_{x1}, p_{x2}, p_{x3}, ..., p_{xk}px1​,px2​,px3​,...,pxk​ 为素数) , y=py1⋅py2⋅py3...pyky = p_{y1} \cdot p_{y2} \cdot p_{y3} ... p_{yk}y=py1​⋅py2​⋅py3​...pyk​ (py1,py2,py3,...,pykp_{y1}, p_{y2}, p_{y3}, ..., p_{yk}py1​,py2​,py3​,...,pyk​ 为素数), 则 n=x⋅y=px1⋅px2⋅px3...pxk⋅py1⋅py2⋅py3...pykn = x \cdot y = p_{x1} \cdot p_{x2} \cdot p_{x3} ... p_{xk} \cdot p_{y1} \cdot p_{y2} \cdot p_{y3} ... p_{yk}n=x⋅y=px1​⋅px2​⋅px3​...pxk​⋅py1​⋅py2​⋅py3​...pyk​, 所有 n=k+1n = k + 1n=k+1 也可以写成至少一个素数的乘积, 即 P(n) 成立.

虚假证明

  • 证明: 所有的马有相同的颜色

  • 目的: 设 P(n) 表示 n 匹马有相同的颜色.

  • 解:

    • 基本情况 (n = 1): 当只有一匹马时, 显然所有的马都有相同的颜色, 所有 P(1) 成立.

    • 归纳假设 (n = k): 假设 ∀n=k\forall n = k∀n=k (k≥1k \geq 1k≥1), P(n) 都成立.

    • 归纳步骤 (n = k + 1):

      对于一个包含 k + 1 匹马的集合 {h1,h2,...,hn+1h_1, h_2, ..., h_{n + 1}h1​,h2​,...,hn+1​}, 如果我们排除最后一匹马, 可以得到一个包含 k 匹马的集合 S1S_1S1​, 根据归纳假设可得 P(k) 成立, 即前 k 匹马的颜色相同, 同样, 如果我们排除第一匹马, 我们可以得到一个包含 k 匹马的集合 S2S_2S2​, 根据归纳假设可得 P(k) 成立, 即后 k 匹马的颜色相同, 因为集合 {h2,...,hnh_2, ..., h_nh2​,...,hn​} 是集合 S1,S2S_1, S_2S1​,S2​ 的交集, 所以可得h1h_1h1​ 和 h2h_2h2​ 颜色相同, 即 k + 1 匹马的颜色都相同, 则 P(k + 1) 成立.

    • 根据数学归纳法的定义可得原命题成立.

  • 该证明的结果完全违背常理, 那么该证明错在哪里呢?

    当 n = 2 时, 排除最后一匹马得到的集合 S1S_1S1​ 和排除第一匹马得到的集合 S2S_2S2​ 没有交集, 此时无法通过交集得出第一匹马和第二匹马的颜色相等, 则归纳步骤不成立, 所以该证明错在无法证明当 n = 2 时 P(n) 成立.
