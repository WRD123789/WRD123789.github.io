QT 资源文件

资源文件

Qt 资源系统是一个跨平台的资源机制, 用于将程序运行时所需要的资源以二进制的形式存储于可执行文件内部. 在程序编译过程中, 添加到资源文件中的文件也会以二进制的形式被打包到可执行程序中, 这样这些资源就永远和可执行程序捆绑到一起了, 不会出现加载资源却找不到的问题.

资源文件中一般添加的都是比较小的资源, 如: 图片、配置文件、MP3 等, 如果是类似视频这样的大文件就不适合放到资源文件中.

使用:

  1. 使用资源编辑器打开资源文件:
  2. 给资源添加前缀 (一个资源文件中可以添加多个前缀, 前缀就是添加的资源在资源文件中的路径, 前缀根据实际需求制定即可, 路径以 / 开头)

  3. 添加文件, 前缀添加完毕, 就可以在某个前缀下边添加相关的资源:

    需要添加的文件必须和项目文件 (.pro) 在同一级目录或在更深的目录中.
  4. 可以给添加的文件设置别名, 设置别名之后原来的名字就不能使用 (在高版本的QT Creator 中, 资源文件名字或者别名不支持中文). 设置别名是为了方便维护代码. 如果使用了别名, 当某个文件的文件名发生更改时, 就不需要在代码中更改所有使用该文件的代码
  5. 使用资源文件: 对于添加到资源文件中的资源, 可以通过 ‘:’ 开头的路径找到这个文件. 比如上面的前缀是 “/images”, 文件是 22.png, 那么就可以通过 :/images/22.png 找到这个文件; 也可以找到需要使用的资源图片节点, 鼠标右键, 弹出的菜单中选择 Copy Path ...
