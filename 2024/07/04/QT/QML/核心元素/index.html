QML 核心元素

核心元素

元素可以分为可见元素和不可见元素. 可见元素 (如矩形) 具有几何形状, 通常在屏幕上显示一个区域. 不可见元素 (如 Timer) 提供通用功能, 通常用于操作可见元素.

Item 元素

Item 是所有可见元素的基元素, 它本身并不绘制任何东西, 而是定义了所有可见元素的共有属性:

  • 几何. x 和 y 定义了元素的左上角位置, width 和 height 定义了元素的大小, z 定义了元素的堆叠顺序, 也就是元素在三维空间中的位置.
  • 布局处理. anchors (left, right, top, bottom, verticalCenter 和 horizontalCenter) 定位元素相对于其他元素的位置, 还可以使用可选的边距 (margins) 来调整元素的位置.
  • 键处理. Key 属性用于处理键盘事件, 你可以在一个元素的 Key 属性中定义各种按键处理函数; KeyNavigation 属性用于控制键盘导航, 你可以在一个元素的 KeyNavigation 属性中定义如何通过键盘的方向键在不同的元素之间切换焦点; focus 用于控制元素是否可以接收键盘输入.
  • 转换. scale (缩放) 属性能够改变元素的尺寸, rotate (旋转) 属性能够围绕一个点旋转元素, transform 属性能够同时改变元素在 x、y 和 z 三个轴上的位置和尺寸, transformOrigin 设置变换原点.
  • 视觉. opacity 属性控制透明度, visible 属性设置元素是否可见, clip 属性用于限制元素的绘制范围, smooth 属性用于提高元素的渲染质量 (当 smooth 属性被设置为 true 时, 元素的边缘将被平滑处理).
  • 状态定义. states 是一个包含了元素所有可能状态的列表属性, 每个状态都是元素的一种特定配置, 它定义了元素在该状态下的所有属性值; currentState 属性用于设置和获取元素的当前状态; transitions 是一个包含了元素所有可能的状态过渡的列表属性。每个过渡定义了元素从一个状态切换到另一个状态时的动画效果.

Rectangle 元素

Rectangle 继承自 Item, 并且可以填充颜色 (在 QML 中, 颜色一般用 RGB 字符串, 如 “#FF4444”, 或者颜色名, 如 “white”). 另外, Rectangle 通过 border.color 和 border.width 来设置边框, 可以通过 radius 创建圆角矩形.

Image 元素

Image 元素能够以各种格式显示图像 (如 PNG、 JPG、 GIF、 BMP、 WEBP). 下面是一些常见的 Image 属性:

属性 描述
source 指定图片路径
mirror 用于控制图像是否镜像显示, 即水平反转
mirrorVertically 用于控制图像是否镜像显示, 即垂直反转
fillMode 用于控制图像的显示方式
asynchronous 用于指定图像加载时是否启用异步模式
status 表示当前图像的加载状态 (图像加载出错时, 状态为 Image.Error)
progress 表示当前图像的加载进度

当源图像的大小与项的大小不同时, 可以通过 fillMode 属性控制图像的显示方式, 来达到缩放和对齐图片以适应 Image 元素的内容矩形 (即可视区域) 的目的.

该属性包含以下几种选项:

选项 描述
Image.Strech (默认值) 缩放图像以填充可视区域, 无需保持宽高比
Image.PreserveAspectFit 保持宽高比, 并缩放图像以适合可视区域
Image.PreserveAspectCrop 保持图像宽高比, 并缩放图像以填充可视区域 (必要时进行裁剪)
Image.Tile 将原始图像平铺在可视区域中 (当图像大小小于可视区域时, 图像会水平和垂直复制)
Image.TileVertically 水平居中重复显示元素, 直到填充整个可视区域 (与 “Tile” 不同, 此选项会垂直平铺, 但在水平方向上只显示一次)
Image.TileHorizontally 垂直居中重复显示元素, 直到填充整个可视区域 (与 “Tile” 不同, 此选项会水平平铺, 但在垂直方向上只显示一次)
Image.Pad 图像未转换
