信息的表示和处理

信息存储

大多数计算机使用 8 位的块, 或者字节 (byte), 作为最小的可寻址的内存单位. 机器级程序将内存视为一个非常大的字节数组, 称为虚拟内存 (Virtual Memory). 内存的每个字节都由一个唯一的数字来标识, 称为地址. 所有可能地址的集合就称为虚拟地址空间 (Virtual Address Space).

虚拟地址空间只是展现给机器级程序的一个假象. 实际的实现是将动
态随机访问存储器 (DRAM)、闪存、磁盘存储器、特殊硬件和操作系统软件结合起来, 为程序提供一个看上去统一的字节数组.

表示字符串

C 语言中字符串被编码为一个以 null (值为 0) 字符结尾的字符数组. 每个字符都由某个标准编号来表示, 最常见的是 ASCII 字符码. 在使用 ASCII 码作为字符码的任何系统上都将得到相同的结果, 与字节顺序和字大小规则无关. 因此文本数据比二进制数据具有更好的平台独立性.

布尔代数

二进制值是计算机编码、存储和操作信息的核心, 所以围绕数值 0 和 1 的研究演化出了布尔代数.

基本布尔代数运算

  • &: 与运算, A = 1 且 B = 1 时, A & B = 1.
  • |: 或运算, A = 1 或 B = 1 时, A | B = 1.
  • ~: 非运算, A = 1 时, ~A = 0; A = 0 时, ~A = 1.
  • ^: 异或运算, A = 1 或 B = 1 时, A ^ B = 1; A = 1 且 B = 1 时, A ^ B = 0.

可以将上述布尔运算扩展到位向量的运算, 位向量就是固定长度为 w、由 0 和 1 组成的串. 位向量的运算可以定义为参数的每个对应元素之间的运算.

假设 a 和 b 分别表示位向量 [aw−1, aw−2, ..., a0][a_{w - 1}, \ a_{w - 2}, \ ..., \ a_0][aw−1​, aw−2​, ..., a0​], [bw−1, bw−2, ..., b0][b_{w - 1}, \ b_{w - 2}, \ ..., \ b_0][bw−1​, bw−2​, ..., b0​], 则将 a & b 定义为一个长度为 w 的位向量, 其中第 i 个元素等于 aia_iai​ & bib_ibi​, 0≤i<w0 \leq i < w0≤i<w. 同样, 可以用类似的方式定义位向量的 | 、~ 、^ 运算.

可以使用位向量来对集合进行编码来表示有限集合. 我们可以用位向量 [aw−1, aw−2, ..., a0][a_{w - 1}, \ a_{w - 2}, \ ..., \ a_0][aw−1​, aw−2​, ..., a0​] 编码任何子集 A ⊆\subseteq⊆ {0, 1, …, w - 1}, 其中 ai=1a_i = 1ai​=1 当且仅当 i∈Ai \in Ai∈A.

例如:

使用这种编码集合的方式, 布尔运算 | 和 & 分别对应于集合的并和交, 而 ~ 对应于集合的补.

对于两个非负整数 x 和 y, 满足 0≤x, y<2w0 \leq x, \ y < 2^w0≤x, y<2w, 即 x, y 为 w 位的无符号数, 如果计算它们的和, 和可能的范围为 0≤x+y≤2w+1−20 \leq x + y \leq 2^{w + 1} - 20≤x+y≤2w+1−2, 此时需要 w + 1 位来保存运算结果, 此时完整的计算结果无法保存, 便会发生溢出.

定义运算 x+wuyx +_w^u yx+wu​y, 该操作是将整数和 x + y 截断为 w 位.

对满足 0≤x, y<2w0 \leq x, \ y < 2^w0≤x, y<2w 的 x 和 y 有:

设 t=2wt = 2^wt=2w

则 x+wuy={x+y,x + y < tx+y−2w,t ≤ x + y < 2tx +_w^u y = \begin{cases} x + y, & \text{x + y < t} \\ x + y - 2^w, & \text{t ≤ x + y < 2t} \end{cases}x+wu​y={x+y,x+y−2w,​x + y < tt ≤ x + y < 2t​

可见当 x+y≥2wx + y \geq 2^wx+y≥2w 时, 会发生溢出, 此时会截断计算结果; 当 x+y<2wx + y < 2^wx+y<2w 时, 值保持不变.

对于在范围 0≤x, y≤UMaxw0 \leq x, \ y \leq UMax_w0≤x, y≤UMaxw​ 中的 x 和 y, 令 s=x+wuys = x +_w^u ys=x+wu​y, 如果 s < x 或 s < y, 则发生溢出.

补码加法

对于在范围 −2w−1≤x, y≤2w−1−1-2^{w - 1} \leq x, \ y \leq 2^{w - 1} - 1−2w−1≤x, y≤2w−1−1 之内的有符号整数 x 和 y, 它们的和就在范围 −2w≤x+y≤2w−2-2^w \leq x + y \leq 2^w - 2−2w≤x+y≤2w−2 之内, 同样计算结果无法完整存储, 使用截断使之达到 w 位.

对于满足 −2w−1≤x, y≤2w−1−1-2^{w - 1} \leq x, \ y \leq 2^{w - 1} - 1−2w−1≤x, y≤2w−1−1 的整数 x 和 y 有:

设 t=2w−1t = 2^{w - 1}t=2w−1

则 x+wty={x+y−2w,t ≤ x + yx+y,-t ≤ x + y < tx+y+2w,x + y < -tx +_w^t y = \begin{cases} x + y - 2^w, & \text{t ≤ x + y} \\ x + y, & \text{-t ≤ x + y < t} \\ x + y + 2^w, & \text{x + y < -t} \end{cases}x+wt​y=⎩⎨⎧​x+y−2w,x+y,x+y+2w,​t ≤ x + y-t ≤ x + y < tx + y < -t​

当 x+y<TMinwx + y < TMin_wx+y<TMinw​ 时, 发生负溢出, 需要在和数中加上 2w2^w2w; 当 x+y>TMaxwx + y > TMax_wx+y>TMaxw​ 时, 发生正溢出, 需要在和数中减去 2w2^w2w.

由于两个数的 w 位补码之和与无符号之和有完全相同的位级表示, 所以可以使用同样的机器指令来执行无符号或有符号加法.

对于满足 TMinw≤x, y≤TMaxwTMin_w \leq x, \ y \leq TMax_wTMinw​≤x, y≤TMaxw​ 的 x 和 y, 令 s=x+wtys = x +_w^t ys=x+wt​y. 当且仅当 x > 0, y > 0 但 s≤0s \leq 0s≤0 时, 发生正溢出; 当且仅当 x < 0, y < 0 但 s≥0s \geq 0s≥0 时, 发生负溢出.

补码的非

对满足 TMinw≤x≤TMaxwTMin_w \leq x \leq TMax_wTMinw​≤x≤TMaxw​ 中的每个数 x, 其补码的非 −wtx={TMin,x = Tmin−x,x > TMin-_w^t x = \begin{cases} TMin, & \text{x = Tmin} \\ -x, & \text{x > TMin} \end{cases}−wt​x={TMin,−x,​x = Tminx > TMin​.

即对于 w 位的补码加法, TMinwTMin_wTMinw​ 是自己的加法的逆 (相反数), 而对于其他数 x 都有 -x 作为其加法的逆.

计算一个位级表示的补码非有两种方法:

  • 对每一位取反然后对取反后的结果加 1, 即 -x = ~x + 1.
  • 假设数 x 的第 k 位代表其位级中最右边的 1 的位置, 则 x 的位级表示为 [xw−1, xw−2, ..., 1, 0, ..., 0][x_{w - 1}, \ x_{w - 2}, \ ..., \ 1, \ 0, \ ..., \ 0][xw−1​, xw−2​, ..., 1, 0, ..., 0] (x≠0x \neq 0x=0), 则这个值的非写成二进制为 [∼xw−1, ∼xw−2, ..., 1, 0, ..., 0][\sim x_{w - 1}, \ \sim x_{w - 2}, \ ..., \ 1, \ 0, \ ..., \ 0][∼xw−1​, ∼xw−2​, ..., 1, 0, ..., 0]

无符号乘法

范围在 0≤x, y≤2w−10 \leq x, \ y \leq 2^w - 10≤x, y≤2w−1 内的整数 x 和 y 可以被表示为 w 位的无符号数, 则 x⋅yx \cdot yx⋅y 的取值范围为 0∼(2w−1)2=22w−2w+1+10 \sim (2^w - 1)^2 = 2^{2w} - 2^{w + 1} + 10∼(2w−1)2=22w−2w+1+1. 需要 2w2w2w 位才能容纳 x⋅yx \cdot yx⋅y 的计算结果.

C 语言中的无符号乘法通常保留 2w 位整数乘积的低 w 位, 这个值表示为 x∗wuyx *_w^u yx∗wu​y.

则对于满足 0≤x, y≤UMaxw0 \leq x, \ y \leq UMax_w0≤x, y≤UMaxw​ 的 x 和 y 有:

x∗wuy=(x⋅y) mod 2wx *_w^u y = (x \cdot y) \ mod \ 2^wx∗wu​y=(x⋅y) mod 2w

补码乘法

范围在 −2w−1≤x, y≤2w−1−1-2^{w - 1} \leq x, \ y \leq 2^{w - 1} - 1−2w−1≤x, y≤2w−1−1 内的整数 x 和 y 可以被表示为 w 位的补码数字, 但是它们的乘积 x⋅yx \cdot yx⋅y 的取值范围为 −22w−2+2w−1∼22w−2-2^{2w - 2} + 2^{w - 1} \sim 2^{2w - 2}−22w−2+2w−1∼22w−2, 此时需要 2w2w2w 位才能容纳 x⋅yx \cdot yx⋅y 的计算结果. 所以 C 语言中的有符号乘法通过将 2w 位的乘积截断为 w 位来实现, 称这个值为 x∗wtyx *_w^t yx∗wt​y.

对于满足 TMinw≤x, y≤TMaxwTMin_w \leq x, \ y \leq TMax_wTMinw​≤x, y≤TMaxw​ 的 x 和 y 有:

乘以常数

在大多数机器上, 整数乘法相当移位、加法等操作时间代价巨大, 于是考虑用移位和加法运算的组合来替代乘以常数因子的乘法.

首先考虑乘以 2 的幂的情况:

设 x 为位模式 [xw−1, xw−2, ..., x0][x_{w - 1}, \ x_{w - 2}, \ ..., \ x_0][xw−1​, xw−2​, ..., x0​] 表示的无符号整数, 那么, 对于任何 k≤0k \leq 0k≤0, x⋅2kx \cdot 2^kx⋅2k 的 w + k 位无符号整数的位模式为 [xw−1, xw−2, ..., x0, 0, ..., 0][x_{w - 1}, \ x_{w - 2}, \ ..., \ x_0, \ 0, \ ..., \ 0][xw−1​, xw−2​, ..., x0​, 0, ..., 0].

当相乘的常数不是 2 的幂时, 需要以移位、加法和减法的组合来替代乘法.

计算机表达式 x * 14, 由于 14 = 23+22+212^3 + 2^2 + 2^123+22+21, 则可将上述表达式重写成 (x << 3) + (x << 2) + (x << 1) 或 (x << 4) - (x << 1) 的形式.

对于 x * K, 常数 K 可以表示为一组 0 和 1 交替的序列:

[(0 … 0) (1…1)(0…0)…(1…1)]

考虑其中的某一组从位置 n 到位置 m 的连续的 1 (n≥mn \geq mn≥m), 可以用以下两种形式来计算:

  • (x << n) + (x << (n - 1)) + … + (x << m)
  • (x << (n + 1)) - (x << m)

可以将每组连续的 1 加起来得到 x * K. 大多数编译器只在需要少量移位、加法和减法时才考虑这种优化.

除以 2 的幂

在大多数机器上, 整数除法比整数乘法更慢, 所以需要将除以 2 的幂用移位来替代.

对于任何实数 a, 定义 ⌊a⌋\lfloor a \rfloor⌊a⌋ 为唯一的整数 a’, 是的 a′≤a<a′+1a' \leq a < a' + 1a′≤a<a′+1.

同理 ⌈a⌉\lceil a \rceil⌈a⌉ 为唯一的整数 a’, 使得 a′−1<a≤a′a' - 1 < a \leq a'a′−1<a≤a′.

整数除法总是舍入到 0, 即当 x≥0x \geq 0x≥0 且 y>0y > 0y>0, 结果为 ⌊x/y⌋\lfloor x / y \rfloor⌊x/y⌋; 对于 x<0x < 0x<0 且 y>0y > 0y>0, 结果为 ⌈x/y⌉\lceil x / y \rceil⌈x/y⌉.

除以 2 的幂的无符号除法

设 x 为位模式 [xw−1, xw−2, .., x0][x_{w - 1},\ x_{w - 2}, \ .., \ x_0][xw−1​, xw−2​, .., x0​] 表示的无符号整数, 而 k 的取值范围为 0≤k<w0 \leq k < w0≤k<w. 设 x’ 为 x - k 位位表示 [xw−1, xw−2, ..., xk][x_{w - 1}, \ x_{w - 2}, \ ..., \ x_k][xw−1​, xw−2​, ..., xk​] 的无符号数, 而 x’’ 为 k 位位表示 [xk−1, xk−2, ..., x0][x_{k - 1}, \ x_{k - 2}, \ ..., \ x_0][xk−1​, xk−2​, ..., x0​] 的无符号数. 可得 x=2k⋅x′+x′′x = 2^k \cdot x' + x''x=2k⋅x′+x′′, 而 0≤x′′<2k0 \leq x'' < 2^k0≤x′′<2k, 所以可得 ⌊x/2k⌋=x′\lfloor x / 2^k \rfloor = x'⌊x/2k⌋=x′ , 即无符号数除以 2 的幂可以转化为逻辑右移 k 位.

除以 2 的幂的补码除法

设 x 为位模式 [xw−1, xw−2, .., x0][x_{w - 1},\ x_{w - 2}, \ .., \ x_0][xw−1​, xw−2​, .., x0​] 表示的无符号整数, 而 k 的取值范围为 0≤k<w0 \leq k < w0≤k<w. 设 x’ 为 x - k 位位表示 [xw−1, xw−2, ..., xk][x_{w - 1}, \ x_{w - 2}, \ ..., \ x_k][xw−1​, xw−2​, ..., xk​] 的无符号数, 而 x’’ 为 k 位位表示 [xk−1, xk−2, ..., x0][x_{k - 1}, \ x_{k - 2}, \ ..., \ x_0][xk−1​, xk−2​, ..., x0​] 的无符号数. 可得 x=2k⋅x′+x′′x = 2^k \cdot x' + x''x=2k⋅x′+x′′, 而 0≤x′′<2k0 \leq x'' < 2^k0≤x′′<2k, 所以可得 ⌊x/2k⌋=x′\lfloor x / 2^k \rfloor = x'⌊x/2k⌋=x′, 由于对 [xw−1, xw−2, .., x0][x_{w - 1},\ x_{w - 2}, \ .., \ x_0][xw−1​, xw−2​, .., x0​] 进行算术右移 k 位得到的位向量 [xw−1, ..., xw−1, xw−2, ..., xk][x_{w - 1}, \ ..., \ x_{w - 1}, \ x_{w - 2}, \ ..., \ x_k][xw−1​, ..., xw−1​, xw−2​, ..., xk​] 正好相当于对位向量 [xw−1, xw−2, ..., xk][x_{w - 1}, \ x_{w - 2}, \ ..., \ x_k][xw−1​, xw−2​, ..., xk​] 从 w - k 位符号扩展到 w 位, 所以直接对补码 x 进行算术右移得到的结果为 ⌊x/2k⌋\lfloor x / 2^k \rfloor⌊x/2k⌋, 这种结果不符合之前对整数除法的定义.

可以通过在移位 k 位前增加一个偏置值 2k−12^k - 12k−1 使得对于负数补码值 x 和无符号数 k (0≤k<w0 \leq k < w0≤k<w), (x+(1<<k)−1)>>k=⌈x/2k⌉(x + (1 << k) - 1) >> k = \lceil x / 2^k \rceil(x+(1<<k)−1)>>k=⌈x/2k⌉.

要证 ⌈x/y⌉=⌊(x+y−1)/y⌋\lceil x / y \rceil = \lfloor (x + y - 1) / y \rfloor⌈x/y⌉=⌊(x+y−1)/y⌋, 假设 x = qy + r, 其中 0≤r<y0 \leq r < y0≤r<y, 得到 (x + y - 1) / y = q + (r + y - 1) / y, 因此 ⌊(x+y−1)/y⌋=q+⌊(r+y−1)/y⌋\lfloor (x + y - 1) / y \rfloor = q + \lfloor (r + y - 1) / y \rfloor⌊(x+y−1)/y⌋=q+⌊(r+y−1)/y⌋. 当 r = 0 时, ⌊(x+y−1)/y⌋=q\lfloor (x + y - 1) / y \rfloor = q⌊(x+y−1)/y⌋=q; 当 r > 0 时, ⌊(x+y−1)/y⌋=q+1\lfloor (x + y - 1) / y \rfloor = q + 1⌊(x+y−1)/y⌋=q+1.

综上, 对于除以 2 的幂的补码除法, 对于正数, 直接算术右移 k 位得到结果; 对于负数, 在算术右移 k 位之前需要添加偏置值 2k−12^k - 12k−1. 用 C 表示为 (x < 0 ? x + (1 << k) - 1 : x) >> k.
