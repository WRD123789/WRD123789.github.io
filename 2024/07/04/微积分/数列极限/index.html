收敛数列性质

定义

  • 若数列{ ana_nan​ } 有极限, 则极限必唯一.
  • 若数列{ ana_nan​ } 有极限, 称{ ana_nan​ } 收敛, 否则称{ ana_nan​ } 发散.

极限不变性

一个数列改变它的有限项或去掉数列前面有限项或在前面添加有限项, 不改变数列的收敛性. 若原来数列收敛, 改变后极限值不变.
