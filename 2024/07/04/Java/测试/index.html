JUnit

JUnit 是一个 Java 编程语言编写的单元测试框架, 为简化测试的编写提供了许多有用的方法和功能.

语法链接: http://junit.org/junit4/javadoc/4.12/org/junit/Assert.html

测试注释:

  • 在每个测试方法前面加上 @org.junit.Test (没有分号).
  • 将每个测试方法更改为为非静态的 (即去掉关键字 static).
  • 从 Test 类中删除 main 方法.

此时运行代码时所有测试都会执行, 而无需手动调用.

导入库: 在文件的顶端添加 import static org.junit.Assert.*, 此时可以省略 org.junit.Assert. , 例如: 此时org.junit.Assert.assertEquals(expected, actual) 可以写成 assertEquals(expected, actual).
