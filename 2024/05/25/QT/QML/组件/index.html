组件

组件是一种可重复使用的元素. QML 提供了创建组件的不同方法, 下面介绍基于文件的组件. 一个基于文件的组件是通过把 QML 元素放在一个文件中, 并给文件取一个元素名 (如 Button.qml) 来创建的. 你可以像使用 Qt Quick 模块中的其他元素一样使用该组件 (该例中为 Button { ... }).

在另一个文件中可以使用该组件:

当点击 Button 组件时, 会自动调用 Button 组件中的 MouseArea 元素的 onClicked 事件处理函数, 在该函数中主动发送 Button 组件的 clicked 信号, 然后会自动调用 Button 组件的 onClicked 事件处理函数 (由外部定义), 在该函数中改变 status 的文本值.
