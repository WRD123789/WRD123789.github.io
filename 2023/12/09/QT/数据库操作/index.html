数据库操作

QT 框架中对数据库操作提供了很好的支持, 我们可以通过 QT 提供的类非常方便地和本地或者远程数据库进行连接.

如果想用通过 QT 访问数据库, 首先需要在项目的构造文件 (以 qmake 为例) 中添加模块名:

这个 sql 模块对应的是一个动态库, 动态库中有数据库相关类的二进制源码, 在使用动态库中的数据库类的时候, 动态库被加载到内存中.

QT 中常用的数据库类如下:

  • QSqlDatabase: 通过这个类添加 / 删除 / 复制 / 关闭数据库实例.
  • QSqlQuery: 数据库查询类, 主要是用来执行 SQL 语句和处理查询结果.
  • QSqlRecord: 用于表示数据库中一条记录的类.
  • QSqlQueryModel: 是 QT 中的一个模型类, 用于将数据库查询的结果集与 QT 的视图 (例如 QTableView) 关联起来.
  • QSqlError: 数据操作失败可以通过这个类获取相关的错误信息.

QSqlDatabase

QSqlDatabase 类提供了一个连接访问数据库的接口. 一个 QSqlDatabase 的实例就表示一个连接, 该连接通过数据库驱动程序提供对数据库的访问, 这些驱动程序派生自 QSqlDriver.
