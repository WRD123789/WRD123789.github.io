性质

二叉树是完备的并且服从最小堆性质:

  • 最小堆性质: 每个节点的键值都小于等于它的两个子节点的键值.
  • 完备性: 只在底层缺失元素, 所有的节点尽量靠左.

建堆时间复杂度

假设该优先队列 (堆) 为满二叉树, 堆高为 h (从 0 开始算), 又设每层的高度 为 hih_ihi​, 每层的节点数为 nin_ini​, 则建堆的时间计算为 t(h) = ∑0h−1 ni⋅hi\sum_{0}^{h - 1} \ ni \cdot hi∑0h−1​ ni⋅hi.
由于第 0 层有 202^020 个节点, 第 1 层有 212^121 个节点 … 所以 nin_ini​ = 2i2^i2i.
由于第 0 层需要向下调整 h 次, 第 1 层需要向下调整 h - 1 次 … 所以 hih_ihi​ = h−ih - ih−i.
所以 t(h) = 20×h+21×(h−1)+...+2h−1×12^0 \times h + 2^1 \times (h - 1) + ... + 2^{h - 1} \times 120×h+21×(h−1)+...+2h−1×1 = 2h+1−2−h2^{h + 1} - 2 - h2h+1−2−h.
假设总共有 N 个元素, 则树高 h=log2N+1h = log_2^{N + 1}h=log2N+1​, 所以 t(h) = 2log2N+1+1−2−log2N+12^{log_2^{N + 1} + 1} - 2 - log_2^{N + 1}2log2N+1​+1−2−log2N+1​, 其中 2log2N+1+12^{log_2^{N + 1} + 1}2log2N+1​+1 ≈ N, 所以 t = N−2−logNN - 2 - log^NN−2−logN, 所以建堆的时间复杂度为 O(N)O(N)O(N).
