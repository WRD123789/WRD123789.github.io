зӘ—еҸЈ

еҶ…еөҢзӘ—еҸЈдёҺзӢ¬з«ӢзӘ—еҸЈ

  • еҶ…еөҢзӘ—еҸЈ
    • дҫқйҷ„дәҺжҹҗдёҖдёӘеӨ§зҡ„зӘ—еҸЈ (еңЁеҲӣе»әзӘ—еҸЈеҜ№иұЎзҡ„ж—¶еҖҷжҢҮе®ҡзҲ¶еҜ№иұЎжқҘдҫқйҷ„), дҪңдёәдәҶеӨ§зӘ—еҸЈзҡ„дёҖйғЁеҲҶ.
    • еӨ§зӘ—еҸЈе°ұжҳҜиҝҷдёӘеҶ…еөҢзӘ—еҸЈзҡ„зҲ¶зӘ—еҸЈ.
    • зҲ¶зӘ—еҸЈжҳҫзӨәзҡ„ж—¶еҖҷ (и°ғз”Ё show еҮҪж•°), еҶ…еөҢзҡ„зӘ—еҸЈд№ҹе°ұиў«жҳҫзӨәеҮәжқҘдәҶ (еҶ…еөҢзӘ—еҸЈеҜ№иұЎдёҚйңҖиҰҒи°ғз”Ё show еҮҪж•°).
  • зӢ¬з«ӢзӘ—еҸЈ (дёҚеҶ…еөҢ)
    • иҝҷзұ»зӘ—еҸЈжңүиҫ№жЎҶ, жңүж Үйўҳж Ҹ.
    • йңҖиҰҒи°ғз”ЁеҮҪж•° (show еҮҪж•°) жүҚеҸҜд»ҘжҳҫзӨә.

еқҗж ҮдҪ“зі»

еңЁ Qt е…ідәҺзӘ—еҸЈзҡ„жҳҫзӨәжҳҜйңҖиҰҒжҢҮе®ҡдҪҚзҪ®зҡ„, иҝҷдёӘдҪҚзҪ®жҳҜйҖҡиҝҮеқҗж ҮжқҘзЎ®е®ҡзҡ„, жүҖжңүеқҗж Үзҡ„йҖүеҸ–еҸҲйғҪжҳҜеҹәдәҺеқҗж ҮеҺҹзӮ№жқҘзЎ®е®ҡзҡ„.

жүҖжңүеқҗж Үзҡ„зЎ®е®ҡйғҪйңҖиҰҒе…ҲжүҫеҲ°еқҗж ҮеҺҹзӮ№, Qt зҡ„еқҗж ҮеҺҹзӮ№еңЁзӘ—еҸЈзҡ„е·ҰдёҠи§’:

  • x иҪҙеҗ‘еҸійҖ’еўһ.
  • y иҪҙеҗ‘дёӢйҖ’еўһ.

еңЁдёҖдёӘ Qt зӘ—еҸЈдёӯдёҖиҲ¬йғҪжңүеҫҲеӨҡеӯҗзӘ—еҸЈеҶ…еөҢеҲ°иҝҷдёӘзҲ¶зӘ—еҸЈдёӯ, е…¶дёӯжҜҸдёӘзӘ—еҸЈйғҪжңүиҮӘе·ұзҡ„еқҗж ҮеҺҹзӮ№:

  • жҜҸдёӘзӘ—еҸЈйғҪжңүеқҗж ҮеҺҹзӮ№, еңЁе·ҰдёҠи§’. еӯҗзӘ—еҸЈзҡ„дҪҚзҪ®жҳҜеҹәдәҺзҲ¶зӘ—еҸЈзҡ„еқҗж ҮдҪ“зі»жқҘзЎ®е®ҡзҡ„, д№ҹе°ұжҳҜиҜҙйҖҡиҝҮзҲ¶зӘ—еҸЈзҡ„еқҗж ҮеҺҹзӮ№жқҘзЎ®е®ҡиҮӘе·ұзҡ„дҪҚзҪ®, жүҖд»ҘзӘ—еҸЈжҳҫзӨәзҡ„ж—¶еҖҷдҪҝз”Ёзҡ„жҳҜзӣёеҜ№еқҗж Ү, зӣёеҜ№дәҺзҲ¶зӘ—еҸЈ.
  • е°ҶеӯҗзӘ—еҸЈз§»еҠЁеҲ°зҲ¶зӘ—еҸЈзҡ„жҹҗдёӘдҪҚзҪ®

    1. е…ідәҺйЎ¶зә§иҸңеҚ•еҸҜд»ҘзӣҙжҺҘеңЁ UI зӘ—еҸЈдёӯеҸҢеҮ», зӣҙжҺҘиҫ“е…Ҙж–Үжң¬дҝЎжҒҜеҚіеҸҜ, еҜ№еә”еӯҗиҸңеҚ•йЎ№д№ҹеҸҜд»ҘйҖҡиҝҮе…ҲеҸҢеҮ»еңЁиҫ“е…Ҙзҡ„ж–№ејҸе®ҢжҲҗж·»еҠ , дҪҶжҳҜиҝҷз§Қж–№ејҸдёҚж”ҜжҢҒдёӯж–Үзҡ„иҫ“е…Ҙ:
    2. дёҖиҲ¬жғ…еҶөдёӢ, жҲ‘д»¬йғҪжҳҜе…ҲеңЁеӨ–йқўеҲӣе»әеҮә QAction еҜ№иұЎ, з„¶еҗҺеҶҚе°Ҷе…¶жӢ–жӢҪеҲ°жҹҗдёӘиҸңеҚ•дёӢиҫ№:
    3. йҖҡиҝҮд»Јз Ғзҡ„ж–№ејҸж·»еҠ иҸңеҚ•жҲ–иҖ…иҸңеҚ•йЎ№:
  • иҸңеҚ•йЎ№ QAction дәӢд»¶зҡ„еӨ„зҗҶ
    еҚ•еҮ»иҸңеҚ•йЎ№, иҜҘеҜ№иұЎдјҡеҸ‘еҮәдёҖдёӘдҝЎеҸ·:

е·Ҙе…·ж Ҹ

  • ж·»еҠ е·Ҙе…·жҢүй’®
    1. е…ҲеҲӣе»ә QAction еҜ№иұЎ, з„¶еҗҺжӢ–жӢҪеҲ°е·Ҙе…·ж Ҹдёӯ, е’Ңж·»еҠ иҸңеҚ•йЎ№зҡ„ж–№ејҸзӣёеҗҢ:
    2. и°ғз”Ёзӣёе…ізҡ„ API еҮҪж•°:
  • еңЁ ui зӘ—еҸЈзҡ„ж ‘зҠ¶еҲ—иЎЁдёӯ, жүҫеҲ°е·Ҙе…·ж ҸиҠӮзӮ№, е°ұеҸҜд»ҘеҲ°зҡ„е·Ҙе…·ж Ҹзҡ„еұһжҖ§и®ҫзҪ®йқўжқҝ:

зҠ¶жҖҒж Ҹ
дёҖиҲ¬жғ…еҶөдёӢ, йңҖиҰҒеңЁзҠ¶жҖҒж Ҹдёӯж·»еҠ жҹҗдәӣжҺ§д»¶, жҳҫзӨәжҹҗдәӣеұһжҖ§, дҪҝз”ЁжңҖеӨҡзҡ„е°ұжҳҜж·»еҠ ж Үзӯҫ QLabel:

дҫӢеҰӮ:

еҒңйқ зӘ—еҸЈ

  • еҒңйқ зӘ—еҸЈеҸҜд»ҘйҖҡиҝҮйј ж ҮжӢ–еҠЁеҒңйқ еҲ°зӘ—еҸЈзҡ„дёҠгҖҒдёӢгҖҒе·ҰгҖҒеҸі, жҲ–иҖ…жө®еҠЁеңЁзӘ—еҸЈдёҠж–№. еҰӮжһңйңҖиҰҒиҝҷз§Қзұ»еһӢзҡ„зӘ—еҸЈеҝ…йЎ»жүӢеҠЁж·»еҠ , еҰӮжһңеңЁйқһ QMainWindow зұ»еһӢзҡ„зӘ—еҸЈдёӯж·»еҠ дәҶеҒңйқ зӘ—еҸЈ, йӮЈд№ҲиҝҷдёӘзӘ—еҸЈжҳҜдёҚиғҪз§»еҠЁе’Ңжө®еҠЁзҡ„.
  • жө®еҠЁзӘ—еҸЈеңЁе·Ҙе…·ж ҸдёӯпјҢ зӣҙжҺҘе°Ҷе…¶жӢ–жӢҪеҲ° ui з•ҢйқўдёҠеҚіеҸҜ:
  • еҒңйқ зӘ—еҸЈд№ҹжңүдёҖдёӘеұһжҖ§йқўжқҝ, жҲ‘д»¬еҸҜд»ҘеңЁе…¶еҜ№еә”еұһжҖ§йқўжқҝдёӯзӣҙжҺҘиҝӣиЎҢи®ҫзҪ®е’Ңдҝ®ж”№зӣёе…іеұһжҖ§:

еёғеұҖ

ж ·ејҸ

QT зӘ—еҸЈеёғеұҖжҳҜжҢҮе°ҶеӨҡдёӘеӯҗзӘ—еҸЈжҢүз…§жҹҗз§ҚжҺ’еҲ—ж–№ејҸеұ•зӨәеҲ°еҜ№еә”зҡ„зҲ¶зӘ—еҸЈдёӯ. еңЁ QT дёӯ, еҰӮжһңеҸӘз”Ёеқҗж ҮжҢҮе®ҡжҺ§д»¶зҡ„дҪҚзҪ®, дёҚдҪҝз”ЁеёғеұҖ, йӮЈд№ҲеҪ“зӘ—еҸЈеӨ§е°ҸеҸ‘з”ҹеҸҳеҢ–ж—¶, е®ғзҡ„еӯҗзӘ—еҸЈзҡ„еӨ§е°ҸдёҚдјҡеҸ‘з”ҹеҸҳеҢ–, дё”жҹҗдәӣйғЁд»¶еҸҜиғҪдјҡиў«е…¶д»–жҺ§д»¶иҰҶзӣ–иҖҢж— жі•жҳҫзӨәеҮәжқҘ.

еңЁ QT дёӯжңүдёүз§Қеёёз”Ёзҡ„еёғеұҖж ·ејҸ:

  • ж°ҙе№іеёғеұҖ: еёғеұҖдёӯзҡ„жүҖжңүзҡ„жҺ§д»¶ж°ҙе№іжҺ’еҲ—.
  • еһӮзӣҙеёғеұҖ: еёғеұҖдёӯзҡ„жүҖжңүзҡ„жҺ§д»¶еһӮзӣҙжҺ’еҲ—.
  • зҪ‘ж јеёғеұҖ: еёғеұҖдёӯзҡ„жүҖжңүзҡ„жҺ§д»¶еһӮзӣҙ+ж°ҙе№іжҺ’еҲ—.

еҗ„з§ҚеёғеұҖжҳҜеҸҜд»Ҙж— йҷҗеөҢеҘ—дҪҝз”Ёзҡ„, иҝҷж ·е°ұеҸҜд»ҘеҲ¶дҪңеҮәеӨҚжқӮзҡ„зӘ—еҸЈ.

и®ҫзҪ®еёғеұҖ

жңүдёүз§ҚеёғеұҖзҡ„ж–№ејҸ:

  1. еңЁ ui зӘ—еҸЈдёӯ, дҪҝз”Ё QT жҸҗдҫӣзҡ„еёғеұҖ, д»Һе·Ҙе…·з®ұдёӯжүҫеҲ°зӣёе…ізҡ„еёғеұҖ, з„¶еҗҺе°Ҷе…¶жӢ–жӢҪеҲ° ui зӘ—еҸЈдёӯ:

    е°Ҷзӣёеә”зҡ„жҺ§д»¶ж”ҫе…ҘеҲ°еёғеұҖеҜ№еә”зҡ„зәўиүІжЎҶеҶ…йғЁ, иҝҷдәӣжҺ§д»¶е°ұдјҡжҢүз…§еёғеұҖзҡ„ж ·ејҸиҮӘеҠЁжҺ’еҲ—:
  2. еңЁ ui зӘ—еҸЈдёӯ, зӣҙжҺҘеңЁзҲ¶зӘ—еҸЈдёӯеҜ№еӯҗйғЁд»¶иҝӣиЎҢеёғеұҖ, еҰӮжһңзӘ—еҸЈз»“жһ„еҫҲеӨҚжқӮйңҖиҰҒеөҢеҘ—, йӮЈд№Ҳе°ұйңҖиҰҒе…Ҳе°ҶиҝҷдәӣеӯҗйғЁд»¶ж”ҫеҲ°дёҖдёӘе®№еҷЁзұ»еһӢзҡ„зӘ—еҸЈдёӯ, з„¶еҗҺеҶҚеҜ№иҝҷдёӘе®№еҷЁзұ»еһӢзҡ„зӘ—еҸЈиҝӣиЎҢеёғеұҖ:



    д№ҹеҸҜд»ҘйҖҡиҝҮзӘ—еҸЈдёҠж–№е·Ҙе…·ж ҸжҸҗдҫӣзҡ„еёғеұҖжҢүй’®иҝӣиЎҢзӘ—еҸЈеёғеұҖ:
  3. еңЁзЁӢеәҸдёӯ, йҖҡиҝҮ API и®ҫзҪ®еёғеұҖ:
    • еңЁ QT дёӯ, еёғеұҖд№ҹжңүеҜ№еә”зҡ„зұ», еёғеұҖзұ»д№Ӣй—ҙзҡ„е…ізі»еҰӮдёӢ:
    • QLayout
    • дёҠиҝ°еёғеұҖзұ»дёӯ, еёёз”Ёзҡ„дёүдёӘеёғеұҖзұ»еҰӮдёӢжүҖзӨә:
      • QHBoxLayout
      • QVBoxLayout
      • QGridLayout

еҪ“жҲ‘д»¬з»ҷзӘ—еҸЈи®ҫзҪ®дәҶеёғеұҖд№ӢеҗҺ, йҖүдёӯеҪ“еүҚзӘ—еҸЈ, е°ұеҸҜд»ҘзңӢеҲ°еңЁе…¶еҜ№еә”зҡ„еұһжҖ§зӘ—еҸЈдёӯйҷӨдәҶжңүзӘ—еҸЈеұһжҖ§, иҝҳжңүдёҖдёӘеёғеұҖеұһжҖ§:

йҖҡиҝҮ ui зј–иҫ‘зӘ—еҸЈзҡ„ж ‘зҠ¶еҲ—иЎЁжҲ‘д»¬еҸҜд»ҘеҜ№жүҖжңүзӘ—еҸЈзҡ„еёғеұҖиҝӣиЎҢжЈҖжҹҘ, еҰӮжһңеҸ‘зҺ°жҹҗдёӘзӘ—еҸЈжІЎжңүеёғеұҖ, дёҖе®ҡиҰҒеҜ№е…¶иҝӣиЎҢи®ҫзҪ®, еҰӮжһңжҹҗдёӘзӘ—еҸЈжІЎжңүиҝӣиЎҢеёғеұҖ, йӮЈд№ҲеҪ“иҝҷдёӘзӘ—еҸЈжҳҫзӨәеҮәжқҘд№ӢеҗҺйҮҢиҫ№зҡ„еӯҗйғЁд»¶е°ұеҸҜиғҪж— жі•иў«жҳҫзӨәеҮәжқҘ:

еј№з°§

еңЁиҝӣиЎҢзӘ—еҸЈеёғеұҖзҡ„ж—¶еҖҷдёәдәҶи®©з•ҢйқўзңӢиө·жқҘжӣҙеҠ зҫҺи§Ӯ, йңҖиҰҒи°ғж•ҙжҺ§д»¶зҡ„дҪҚзҪ®, жҜ”еҰӮ: йқ е·Ұ, йқ еҸі, еұ…дёӯ, еҸҲжҲ–иҖ…жҲ‘д»¬йңҖиҰҒи°ғиҠӮдёӨдёӘжҺ§д»¶д№Ӣй—ҙзҡ„и·қзҰ», д»ҘдёҠиҝҷдәӣйңҖжұӮдҪҝз”Ёеј№з°§йғҪжҳҜеҸҜд»Ҙе®һзҺ°зҡ„.

еј№з°§зҡ„ж ·ејҸжңүдёӨз§Қ:

  • ж°ҙе№іеј№з°§: еңЁж°ҙе№іж–№еҗ‘дёҠиө·дҪңз”Ё.
  • еһӮзӣҙеј№з°§: еңЁеһӮзӣҙж–№еҗ‘дёҠиө·дҪңз”Ё.

еј№з°§д№ҹжңүеҜ№еә”зҡ„еұһжҖ§еҸҜд»Ҙи®ҫзҪ®, е…·дҪ“еұһжҖ§еҰӮдёӢеӣҫжүҖзӨә:

еј№з°§зҡ„ sizeType еұһжҖ§дёӯеёёз”Ёзҡ„дёӨдёӘдёә:

  • Fixed: еҫ—еҲ°дёҖдёӘеӣәе®ҡеӨ§е°Ҹзҡ„еј№з°§.
  • Expanding: еҫ—еҲ°дёҖдёӘеҸҜдјёзј©зҡ„еј№з°§, й»ҳи®Өжғ…еҶөдёӢеј№з°§дјёеұ•еҲ°жңҖеӨ§.
