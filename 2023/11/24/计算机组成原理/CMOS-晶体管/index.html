CMOS жҷ¶дҪ“з®Ў

CMOS жҷ¶дҪ“з®Ўдё»иҰҒжңүдёӨз§Қ:

  • n еһӢеңәж•Ҳеә”жҷ¶дҪ“з®Ў (nFET):

  • p еһӢеңәж•Ҳеә”жҷ¶дҪ“з®Ў (pFET):

з”ұдёҠеҸҜи§Ғ, жҷ¶дҪ“з®ЎеҸҜд»ҘжҠҪиұЎдёәејҖе…і, жү“ејҖж—¶ејҖи·Ҝ (е®һйҷ…дёҠжңүе°‘йҮҸз”өжөҒ), й—ӯеҗҲж—¶зҹӯи·Ҝ (е®һйҷ…дёҠжңүе°‘йҮҸз”өйҳ»).

еӨҡдёӘ CMOS жҷ¶дҪ“з®ЎиҝһжҺҘ:

  • nFET дёІиҒ”

    x=0 iff a AND b = 1

  • nFET е№¶иҒ”

    x=0 if a OR b = 1

  • pFET дёІиҒ”

    x=1 iff a AND b = 0

  • pFET е№¶иҒ”

    x=1 if a OR b = 0

  • еҸҚзӣёеҷЁ

    If in = 0 then out = 1

    If in = 1 then out = 0

  • NAND

  • NOR

CMOS з”өи·Ҝзҡ„е»¶иҝҹ:
з”ұдәҺз”өи·Ҝдёӯзҡ„жүҖжңүиҠӮзӮ№йғҪжңүз”өе®№, дёәдәҶж”№еҸҳз”өе®№зҡ„з”өеҺӢ, йңҖиҰҒе……ж”ҫз”ө, иҖҢе……ж”ҫз”өйңҖиҰҒдёҖе®ҡж—¶й—ҙ,
дё”е……ж”ҫз”өжүҖиҠұиҙ№зҡ„ж—¶й—ҙдёҺз”өйҳ»е’Ңз”өе®№зҡ„еӨ§е°Ҹжңүе…і (П„В вҲјВ RвӢ…C\tau \ \sim \ R \cdot CП„В вҲјВ RвӢ…C).

жүҖд»Ҙ, еҜ№дәҺжҜҸдёӘйҖ»иҫ‘й—Ё, е»¶иҝҹжқҘиҮӘиҫ“е…ҘеҸҳеҢ–еҲ°иҫ“еҮәеҸҳеҢ–.

е»¶иҝҹзҡ„еӨ§е°ҸеҸ–еҶідәҺ:

  • йҖ»иҫ‘й—Ёзҡ„зұ»еһӢ.
  • иҝһжҺҘдәҶеӨҡе°‘йҖ»иҫ‘й—Ё (е»¶иҝҹдјҡзҙҜз§Ҝ, жҹҗдёҖйҖ»иҫ‘й—Ёзҡ„иҫ“еҮәжҳҜеҸҰдёҖйҖ»иҫ‘й—Ёзҡ„иҫ“е…Ҙ, еҚіеҪ“еүҚйҖ»иҫ‘й—Ёиҫ“еҮәдә§з”ҹзҡ„е»¶иҝҹдјҡеҪұе“Қе…¶д»–йҖ»иҫ‘й—Ёзҡ„иҫ“е…Ҙ).
  • е…¶д»–вҖҰ
