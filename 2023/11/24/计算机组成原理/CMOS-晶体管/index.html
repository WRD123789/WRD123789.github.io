CMOS 晶体管

CMOS 晶体管主要有两种:

  • n 型场效应晶体管 (nFET):

  • p 型场效应晶体管 (pFET):

由上可见, 晶体管可以抽象为开关, 打开时开路 (实际上有少量电流), 闭合时短路 (实际上有少量电阻).

多个 CMOS 晶体管连接:

  • nFET 串联

    x=0 iff a AND b = 1

  • nFET 并联

    x=0 if a OR b = 1

  • pFET 串联

    x=1 iff a AND b = 0

  • pFET 并联

    x=1 if a OR b = 0

  • 反相器

    If in = 0 then out = 1

    If in = 1 then out = 0

  • NAND

  • NOR

CMOS 电路的延迟:
由于电路中的所有节点都有电容, 为了改变电容的电压, 需要充放电, 而充放电需要一定时间,
且充放电所花费的时间与电阻和电容的大小有关 (τ ∼ R⋅C\tau \ \sim \ R \cdot Cτ ∼ R⋅C).

所以, 对于每个逻辑门, 延迟来自输入变化到输出变化.

延迟的大小取决于:

  • 逻辑门的类型.
  • 连接了多少逻辑门 (延迟会累积, 某一逻辑门的输出是另一逻辑门的输入, 即当前逻辑门输出产生的延迟会影响其他逻辑门的输入).
  • 其他…
