жңҜиҜӯ

  • CPU: Central ProcessingUnit, дёӯеӨ®еӨ„зҗҶеҷЁ, жҳҜи®Ўз®—жңәзЎ¬д»¶зҡ„ж ёеҝғйғЁд»¶, дё»иҰҒз”ұиҝҗз®—еҷЁе’ҢжҺ§еҲ¶еҷЁз»„жҲҗ.
  • PC: Program Counter, зЁӢеәҸи®Ўж•°еҷЁ, е…¶еҠҹиғҪжҳҜеӯҳж”ҫеҪ“еүҚж¬Іжү§иЎҢжҢҮд»Өзҡ„ең°еқҖ, е№¶еҸҜиҮӘеҠЁи®Ўж•°еҪўжҲҗдёӢдёҖжқЎжҢҮд»Өең°еқҖ.
  • IR: Instruction Register, жҢҮд»ӨеҜ„еӯҳеҷЁ, е…¶еҠҹиғҪжҳҜеӯҳж”ҫеҪ“еүҚжӯЈеңЁжү§иЎҢзҡ„жҢҮд»Ө.
  • CU: Control Unit, жҺ§еҲ¶еҚ•е…ғ(йғЁд»¶), дёәжҺ§еҲ¶еҷЁзҡ„ж ёеҝғйғЁд»¶, е…¶еҠҹиғҪжҳҜдә§з”ҹеҫ®ж“ҚдҪңе‘Ҫд»ӨеәҸеҲ—.
  • ALU: Arithmetic Logic Unit, з®—жңҜйҖ»иҫ‘иҝҗз®—еҚ•е…ғ, дёәиҝҗз®—еҷЁзҡ„ж ёеҝғйғЁд»¶, е…¶еҠҹиғҪжҳҜиҝӣиЎҢз®—жңҜгҖҒйҖ»иҫ‘иҝҗз®—.
  • ACC: Accumulator, зҙҜеҠ еҷЁ, жҳҜиҝҗз®—еҷЁдёӯж—ўиғҪеӯҳж”ҫиҝҗз®—еүҚзҡ„ж“ҚдҪңж•°, еҸҲиғҪеӯҳж”ҫиҝҗз®—з»“жһңзҡ„еҜ„еӯҳеҷЁ.
  • MQ: Multiplier-QuotientRegister, д№ҳе•ҶеҜ„еӯҳеҷЁ, д№ҳжі•иҝҗз®—ж—¶еӯҳж”ҫд№ҳж•°гҖҒйҷӨжі•ж—¶еӯҳж”ҫе•Ҷзҡ„еҜ„еӯҳеҷЁ.
  • X: жӯӨеӯ—жҜҚжІЎжңүдё“жҢҮзҡ„зј©еҶҷеҗ«д№ү, еҸҜд»Ҙз”ЁдҪңд»»дёҖйғЁд»¶еҗҚ, еңЁжӯӨиЎЁзӨәж“ҚдҪңж•°еҜ„еӯҳеҷЁ, еҚіиҝҗз®—еҷЁдёӯе·ҘдҪңеҜ„еӯҳеҷЁд№ӢдёҖ, з”ЁжқҘеӯҳж”ҫж“ҚдҪңж•°пјӣ
  • MAR: Memory AddressRegister, еӯҳеӮЁеҷЁең°еқҖеҜ„еӯҳеҷЁ, еңЁдё»еӯҳдёӯз”ЁжқҘеӯҳж”ҫж¬Іи®ҝй—®зҡ„еӯҳеӮЁеҚ•е…ғзҡ„ең°еқҖ.
  • MDR: Memory Data Register, еӯҳеӮЁеҷЁж•°жҚ®зј“еҶІеҜ„еӯҳеҷЁ, еңЁдё»еӯҳдёӯз”ЁжқҘеӯҳж”ҫд»ҺжҹҗеҚ•е…ғиҜ»еҮәгҖҒжҲ–иҰҒеҶҷе…ҘжҹҗеӯҳеӮЁеҚ•е…ғзҡ„ж•°жҚ®.
  • I/O: Input/Outputequipment, иҫ“е…Ҙ/иҫ“еҮәи®ҫеӨҮ, дёәиҫ“е…Ҙи®ҫеӨҮе’Ңиҫ“еҮәи®ҫеӨҮзҡ„жҖ»з§°, з”ЁдәҺи®Ўз®—жңәеҶ…йғЁе’ҢеӨ–з•ҢдҝЎжҒҜзҡ„иҪ¬жҚўдёҺдј йҖҒ.
  • MIPS: Million Instruction PerSecond, жҜҸз§’жү§иЎҢзҷҫдёҮжқЎжҢҮд»Өж•°, дёәи®Ўз®—жңәиҝҗз®—йҖҹеәҰжҢҮж Үзҡ„дёҖз§Қи®ЎйҮҸеҚ•дҪҚ.
  • е‘Ҫдёӯ: еңЁзј“еӯҳдёӯжүҫеҲ°ең°еқҖ.
  • жңӘе‘Ҫдёӯ: жңӘеңЁзј“еӯҳдёӯжүҫеҲ°ең°еқҖ.
  • еҲ·ж–°: е°ҶжүҖжңүзј“еӯҳиЎҢж Үи®°дёәж— ж•Ҳ, е°Ҷзј“еӯҳдёӯзҡ„жүҖжңүи„Ҹзј“еӯҳиЎҢеҶҷе…ҘеҶ…еӯҳдёӯ.
  • е‘ҪдёӯзҺҮ: еңЁзј“еӯҳдёӯе®һйҷ…е‘Ҫдёӯзҡ„и®ҝй—®зҡ„жҜ”дҫӢ.
  • жңӘе‘ҪдёӯзҺҮ: 1 - е‘ҪдёӯзҺҮ.
  • жңӘе‘Ҫдёӯжғ©зҪҡ: е°Ҷеқ—д»ҺеӯҳеӮЁеұӮж¬Ўзҡ„иҫғдҪҺзә§еҲ«дёӯеҠ иҪҪеҲ°зј“еӯҳдёӯжүҖйңҖиҰҒзҡ„ж—¶й—ҙ.
  • е‘Ҫдёӯж—¶й—ҙ: и®ҝй—®зј“еӯҳжүҖиҠұиҙ№зҡ„ж—¶й—ҙ.
  • жңӘе‘Ҫдёӯж—¶й—ҙ: жңӘе‘Ҫдёӯжғ©зҪҡ + е‘Ҫдёӯж—¶й—ҙ.
  • зј©еҶҷ: з”Ё вҖҳ$вҖҷ д»ЈиЎЁзј“еӯҳ.
  • е№іеқҮеҶ…еӯҳи®ҝй—®ж—¶й—ҙ (AMAT):

    • е‘Ҫдёӯж—¶й—ҙ + жңӘе‘ҪдёӯзҺҮ Г— жңӘе‘Ҫдёӯжғ©зҪҡ.

    • еҜ№дәҺдёҖдёӘ n-зә§ зј“еӯҳзі»з»ҹ:

      AMAT = Time for a L1L_1L1вҖӢ$ hit + (Local) Miss rate L1L_1L1вҖӢ$ Г— (Time for a L2L_2L2вҖӢ$ hit + (Local) Miss rate Г— вҖҰ (Time for a LnL_nLnвҖӢ$ hit + (Local) Miss rate LnL_nLnвҖӢ$ Г— LnL_nLnвҖӢ$ Miss penalty))

  • зј“еӯҳзә§еҲ«: еӯҳеӮЁеұӮж¬Ўз»“жһ„дёӯзҡ„йЎәеәҸ, е…¶дёӯL1$ жҳҜжңҖжҺҘиҝ‘еӨ„зҗҶеҷЁзҡ„, L1 зј“еӯҳеҸӘиғҪеҢ…еҗ«ж•°жҚ® (ж•°жҚ®зј“еӯҳ, D$) жҲ–жҢҮд»Ө (жҢҮд»Өзј“еӯҳ, I$), иҖҢеӨ§еӨҡж•° L2+ зј“еӯҳйғҪжҳҜз»ҹдёҖзҡ„, еҸҜд»ҘеҢ…еҗ«жҢҮд»ӨжҲ–ж•°жҚ®.
  • зј“еӯҳе®№йҮҸ: зј“еӯҳзҡ„жҖ»еӯ—иҠӮж•° (дёҚеҢ…еҗ«ж Үи®°дҪҚе’Ңжңүж•ҲдҪҚ), зј“еӯҳе®№йҮҸ = е…іиҒ”жҖ§ Г— йӣҶеҗҲж•° Г— зј“еӯҳиЎҢеӨ§е°Ҹ.
  • зј“еӯҳиЎҢжҲ–зј“еӯҳеқ—: зј“еӯҳдёӯзҡ„еҚ•дёӘжқЎзӣ®, жҳҜзј“еӯҳдёӯзҡ„еҹәжң¬еӯҳеӮЁеҚ•е…ғ.
  • зј“еӯҳиЎҢеӨ§е°ҸжҲ–еқ—еӨ§е°Ҹ: жҜҸдёҖдёӘзј“еӯҳиЎҢзҡ„еӯ—иҠӮж•°, жҢҮе…¶еҢ…еҗ«зҡ„ж•°жҚ®еқ—зҡ„еӨ§е°Ҹ, дёҚеҢ…еҗ«жңүж•ҲдҪҚе’Ңж Үи®°дҪҚ.
  • зј“еӯҳиЎҢж•°: зј“еӯҳе®№йҮҸ / зј“еӯҳиЎҢеӨ§е°Ҹ.
  • зј“еӯҳе…іиҒ”еәҰ:
    • з»ҷе®ҡең°еқҖеҸҜиғҪеӯҳеңЁзҡ„зј“еӯҳиЎҢж•°.
    • жЈҖжҹҘзј“еӯҳжҳҜеҗҰе‘ҪдёӯжүҖйңҖзҡ„жҜ”иҫғж¬Ўж•°.
    • зӣҙжҺҘжҳ е°„: ж•°жҚ®е…ғзҙ еҸӘжңүдёҖз§ҚеҸҜиғҪзҡ„дҪҚзҪ®.
    • N и·ҜйӣҶеҗҲе…іиҒ”жҳ е°„: ж•°жҚ®е…ғзҙ еҸҜд»ҘдҪҚдәҺ N дёӘдҪҚзҪ®дёӯзҡ„дёҖдёӘ.
    • е…Ёе…іиҒ”жҳ е°„: ж•°жҚ®е…ғзҙ еҸҜд»ҘеңЁзј“еӯҳдёӯзҡ„д»»дҪ•дҪҚзҪ®.
  • зј“еӯҳйӣҶеҗҲж•°: зј“еӯҳиЎҢж•° / зј“еӯҳе…іиҒ”еәҰ
  • е…ЁеұҖжңӘе‘ҪдёӯзҺҮ:
    • еӨҡзә§зј“еӯҳдёӯзҡ„жҹҗдёҖзә§зј“еӯҳзӣёеҜ№дәҺжҖ»зҡ„еҶ…еӯҳи®ҝй—®ж¬Ўж•°зҡ„жңӘе‘ҪдёӯзҺҮ.
    • е…ЁеұҖжңӘе‘ҪдёӯзҺҮ = иҜҘзә§зј“еӯҳзҡ„жңӘе‘Ҫдёӯж¬Ўж•° / CPU и®ҝй—®еҶ…еӯҳзҡ„жҖ»ж¬Ўж•°.
    • е…ЁеұҖжңӘе‘ҪдёӯзҺҮLn_{L_n}LnвҖӢвҖӢ = еұҖйғЁжңӘе‘ҪдёӯзҺҮL1_{L_1}L1вҖӢвҖӢ Г— вҖҰ Г— еұҖйғЁжңӘе‘ҪдёӯзҺҮLn_{L_n}LnвҖӢвҖӢ.
  • еұҖйғЁжңӘе‘ҪдёӯзҺҮ:
    • еӨҡзә§зј“еӯҳдёӯжҹҗдёҖзә§зј“еӯҳзӣёеҜ№дәҺеҲ°иҫҫиҜҘзә§зј“еӯҳзҡ„и®ҝй—®ж¬Ўж•°зҡ„жңӘе‘ҪдёӯзҺҮ.
    • еұҖйғЁжңӘе‘ҪдёӯзҺҮ = иҜҘзә§зј“еӯҳзҡ„жңӘе‘Ҫдёӯж¬Ўж•° / еҲ°иҫҫиҜҘзә§зј“еӯҳзҡ„и®ҝй—®ж¬Ўж•°.
    • еұҖйғЁжңӘе‘ҪдёӯзҺҮLn_{L_n}LnвҖӢвҖӢ = е…ЁеұҖжңӘе‘ҪдёӯзҺҮLn_{L_n}LnвҖӢвҖӢ / е…ЁеұҖжңӘе‘ҪдёӯзҺҮLnвҲ’1_{L_{n-1}}LnвҲ’1вҖӢвҖӢ
  • ж ёеҝғ: д№ҹз§°еҶ…ж ё, жҳҜдёҖдёӘзӢ¬з«Ӣзҡ„жү§иЎҢеҚ•е…ғ, ж №жҚ®еӨ„зҗҶеҷЁзҡ„дёҚеҗҢ, еҸҜд»ҘдёҖж¬ЎиҝҗиЎҢ N дёӘзәҝзЁӢ (дҫӢеҰӮжҷ®йҖҡзҡ„ Intel е’Ң AMD иҠҜзүҮжҜҸдёӘж ёжңүдёӨдёӘзәҝзЁӢ, Raspberry Pi 4 жҜҸдёӘж ёеҸӘжңүдёҖдёӘзәҝзЁӢ).
  • иҝӣзЁӢ: дёҖж®өзЁӢеәҸзҡ„жү§иЎҢиҝҮзЁӢ, иҝӣзЁӢжҳҜдёҖдёӘзӢ¬з«Ӣзҡ„еӯҳеӮЁз©әй—ҙ, з”ЁиҷҡжӢҹеӯҳеӮЁз©әй—ҙеҲҶйҡ”е®ғ, зәҝзЁӢеңЁиҝӣзЁӢдёӯиҝӣиЎҢ, дёҖдёӘиҝӣзЁӢиҮіе°‘жӢҘжңүдёҖдёӘжү§иЎҢзәҝзЁӢ. еңЁдёІиЎҢеӨ„зҗҶеҷЁдёӯ, дёҖдёӘиҝӣзЁӢиҝҗиЎҢдёҖдёӘжү§иЎҢзәҝзЁӢ; еңЁеӨҡеӨ„зҗҶеҷЁдёӯ, еңЁеҗҢдёҖдёӘиҝӣзЁӢдёӯиҝҗиЎҢеӨҡдёӘзәҝзЁӢ.
  • зәҝзЁӢ: жү§иЎҢдёҖдәӣд»»еҠЎзҡ„йЎәеәҸжҢҮд»ӨжөҒ, еӨҡдёӘзәҝзЁӢеҚ з”ЁзӣёеҗҢзҡ„иҝӣзЁӢе’ҢзӣёеҗҢзҡ„еӯҳеӮЁз©әй—ҙ, е°ҶиҪҜд»¶зәҝзЁӢжҳ е°„еҲ°еҸҜз”Ёзҡ„зЎ¬д»¶зәҝзЁӢеҸ–еҶідәҺж“ҚдҪңзі»з»ҹ.
  • ж“ҚдҪңзі»з»ҹзәҝзЁӢ:
    • еҹәдәҺзЎ¬д»¶зәҝзЁӢеӨҚз”ЁеӨҡдёӘиҪҜд»¶зәҝзЁӢ.
    • йҖҡиҝҮеңЁж—¶й—ҙдёҠеӨҚз”ЁеӨҡдёӘзЎ¬д»¶зәҝзЁӢ, з»ҷдёҺдҪ дёҖз§Қжңүи®ёеӨҡзәҝзЁӢеңЁеҗҢж—¶жҙ»и·ғзҡ„еҒҮиұЎ.
    • еңЁдёӯж–ӯзәҝзЁӢж—¶, е°Ҷе®ғзҡ„еҜ„еӯҳеҷЁе’ҢзЁӢеәҸи®Ўж•°еҷЁдҝқеӯҳеҲ°еҶ…еӯҳдёӯ.
    • еңЁжҒўеӨҚзәҝзЁӢж—¶, еҠ иҪҪеҜ№еә”зҡ„еҜ„еӯҳеҷЁе№¶и·іиҪ¬еҲ°еҜ№еә”зҡ„зЁӢеәҸи®Ўж•°еҷЁдёҠ.
  • зЎ¬д»¶еӨҡзәҝзЁӢ:
    • еңЁзј“еӯҳжңӘе‘Ҫдёӯж—¶, зЎ¬д»¶еҲҮжҚўзәҝзЁӢд»ҘеңЁзӯүеҫ…зј“еӯҳжңӘе‘Ҫдёӯзҡ„ж—¶еҖҷеҒҡе…¶д»–е·ҘдҪң.
    • ж”ҫе…ҘеҶ—дҪҷзЎ¬д»¶ (еӨҡдёӘзЁӢеәҸи®Ўж•°еҷЁгҖҒеӨҡдёӘеҜ„еӯҳеҷЁе Ҷзӯү), иҝҷж ·е°ұж— йңҖдҝқеӯҳжҜҸдёӘзәҝзЁӢзҡ„дёҠдёӢж–Ү.
  • RAM:
    • йҡҸжңәеӯҳеҸ–еӯҳеӮЁеҷЁ, д№ҹеҸ«дё»еӯҳ, иғҪеҢ…еҗ«д»Јз Ғе’Ңж•°жҚ®.
    • дёәдәҶи®ҝй—®ж•°жҚ®жҲ–д»Јз Ғ, йңҖиҰҒе°Ҷе®ғд»¬д»ҺзЎ¬зӣҳдёӯеҠ иҪҪеҲ° RAM дёӯ.
  • йЎөйқў:

    • дёәдәҶдҪҝзЈҒзӣҳе’Ң RAM д№Ӣй—ҙзҡ„ж•°жҚ®дј иҫ“жӣҙе®№жҳ“, жҲ‘д»¬еҜ№з§»еҠЁж•°жҚ®зҡ„зІ’еәҰ (д№ҹз§°дёәйЎөйқў) иҝӣиЎҢж ҮеҮҶеҢ–.

    • йЎөйқўзҡ„еӨ§е°ҸйҖҡеёёдёә 4 KB.

    • иҝҷж„Ҹе‘ізқҖеҰӮжһңжҲ‘д»¬жғіиҰҒи®ҝй—®жӯҘйӘӨ RAM дёӯзҡ„ж•°жҚ®, йңҖиҰҒе°Ҷж•°жҚ®жүҖеңЁзҡ„ж•ҙдёӘйЎөйқўеҠ иҪҪеҲ° RAM дёӯ.
