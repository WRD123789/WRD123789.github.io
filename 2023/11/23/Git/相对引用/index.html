相对引用

Git 还提供了相对引用, 即以某一个分支指针指向的提交为参照:

  • 操作符 “^”: 把 ^ 加到引用名称的后面, 表示让 Git 寻找指定分支名指向的提交记录的父节点, 可以引用多个.

    例如: main^ 相当于寻找 “main 的父节点”, main^^ 相当于寻找 “main 的父节点的父节点”.

  • 操作符 “~”: 把 ~<num> 加到引用名称的后面, 表示让 Git 寻找指定分支名指向的提交计量的第 num 个父节点, 没有数字默认第一个父节点.

    例如: main~4, 表示寻找 “main 的第四个父节点”.
