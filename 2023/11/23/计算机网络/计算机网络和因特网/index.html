计算机网络和因特网

因特网

可以从两方面来描述因特网:

  1. 因特网的具体构成, 即构成因特网的基本硬件和软件组件 (组成角度).
  2. 将其视为为分布式应用提供服务的联网基本设施 (功能角度).

具体构成描述

因特网是一个世界范围的计算机网络, 即它是一个互联了遍及全世界数十亿计算设备的网络.

传统设备 (如桌面 PC、Linux 工作站和服务器)和非传统设备 (如智能手机、平板电脑、电视等) 都被称为主机 (host) 或端系统 (end system).

端系统通过通信链路 (communication link) 和分组交换机 (packet switch) 连接在一起.

  • 通信链路由不同类型的物理媒介 (如同轴电缆、铜线、光纤等) 组成, 不同的链路能够以不同的速率传输数据.
  • 当一台端系统要向另一台端系统发送数据时, 发送端系统将数据分段, 并为每段加上首部字节. 由此形成的信息包称为分组. 这些分组通过网络发送到目的端系统, 在那里被装配成原始数据.
  • 分组交换机从它的一条入通信链路接收到达的分组, 并从它的一条出通信链路转发该分组. 两种最著名的分组交换机是路由器 (router) 和链路层交换机 (link-layer switch). 链路层交换机通常用于接入网中, 而路由器通常用于网络核心中. ^8c266b
  • 从发送端系统到接收端系统, 一个分组所经历的一系列通信链路和分组交换机称为通过该网络的路径 (path/route).

端系统通过因特网服务提供商 (Internet Service Provider, ISP) 接入因特网.

  • ISP 包括本地电缆或电话公司那样的住宅区 ISP、公司 ISP、大学 ISP, 机场、旅馆等公共场所提供 WiFi 接入的 ISP, 以及为智能手机等设备提供移动接入的蜂窝数据 ISP.

每个 ISP 自身就是一个由多台分组交换机和多段通信链路组成的网络.

因特网就是将端系统彼此互联, 因此为端系统提供接入的 ISP 必须互联.

较低层的 ISP 通过国家的、国际的较高层 ISP 互联起来, 较高层 ISP 是由通过高速光纤链路互联的高速路由器组成的. 无论是较高层还是较低层 ISP 网络, 它们每个都是独立管理的, 运行着 IP 协议, 遵从一定的命名和地址规则.

端系统、分组交换机和其他因特网部件都要运行一系列协议 (protocol), 这些协议控制因特网中信息的接收和发送. 其中 TCP (Transmission Control Protocol, 传输控制协议) 和 IP (Internet Protocol, 网际协议) 是因特网中最为重要的两个协议. IP 协议定义了在路由器和端系统之间发送和接收的分组格式. 因特网的主要协议统称为 TCP/IP.

服务描述

我们也可以从为应用程序提供服务的基础设施的角度来描述因特网.

涉及多个相互交互数据的端系统的应用程序称为分布式应用程序 (distributed application).

  • 如电子邮件、即时讯息、多人游戏和视频会议等.

与因特网相连的端系统提供了一个套接字接口 (socket interface), 该接口规定了运行在一个端系统上的程序请求因特网基础设施向另一个端系统上的特定目的地程序交付数据的方式. 因特网套接字接口是一套发送程序必须遵循的规则集合, 因此因特网能够将数据交付给目的地.

因特网向应用程序提供了多种服务, 当你研发一种因特网应用程序时, 也必须为你的应用程序选择其中的一种因特网服务.

协议

协议 (protocol) 定义了在两个或多个通信实体之间交换的报文的格式和顺序, 以及发送报文和接收报文时所采取的动作.

因特网广泛地使用了协议, 不同的协议用于完成不同的通信任务.

  • 例如, 当你向一个 Web 服务器发出请求 (即你在 Web 浏览器中键入一个 Web 网页的 URL) 时, 首先你的计算机将向该 Web 服务器发送一条连接请求报文, 并等待回答. 该 Web 服务器如果能接收到连接请求报文, 将返回一条连接响应报文. 得知能正常请求该 Web 网页后, 你的计算机则在一条 GET 报文中发送要从这台 Web 服务器上取回的网页的名字. 最后, Web 服务器向计算机返回该 Web 网页.

网络边缘

通常把与因特网相连的计算机和其他设备称为端系统, 因为它们位于互联网的边缘.

端系统有时候也称为主机, 因为它们容纳应用程序. 而主机又被划分为两类: 客户 (client) 和服务器 (server).

  • 客户通常是桌面 PC、移动 PC 和智能手机等.
  • 服务器通常是更加强大的机器, 用于存储和发布 Web 页面、流视频、中断电子邮件等. 大部分提供搜索结果、电子邮件、Web 页面和视频的服务器都属于大型数据中心 (data center), 即数据中心是由大量服务器组成的.

接入网

接入网是指将端系统物理连接到其边缘路由器的网络, 负责将用户设备的数据传输到核心网络或从核心网络传送数据到用户设备.

  • 边缘路由器是端系统到任何其他远程端系统的路径上的第一台路由器.

以下是几种类型的接入链路:

家庭接入: DSL、电缆、FTTH、拨号和卫星

DSL 因特网接入

数字用户线 (Digital Subscriber Line) 是宽带住宅接入的最流行类型之一. 住宅通常从提供本地电话接入的本地电话公司处获得 DSL 因特网接入, 此时, 用户的本地电话公司也是它的 ISP.

如图所示:

  • 每个用户的 DSL 调制解调器使用现有的电话线与位于电话公司的本地中心局 (CO) 中的数字用户线接入复用器 (DSLAM) 交换数据.
  • 在得到用户设备传递的数据信号后, 家庭端的 DSL 调制解调器将其转换为高频音 (模拟信号) 并通过电话线传递给本地中心局.
  • 最终来自许多家庭的模拟信号在 DSLAM处被重新转换为数字信号.

此时家庭电话线同时承载了数据和传统的电话信号, 它们用不同的频率进行编码:

  • 高速下行信道 (下行指的是数据从网络传输到用户设备), 位于 50kHz 到 1MHz 频段.
  • 中速上行信道 (上行指的是数据从用户设备传输到网络), 位于 4kHz 到 50kHz 频段.
  • 普通的双向电话信道, 位于 0 到 4kHz 频段.

这种方式使得单根 DSL 线路看起来就像有 3 根单独的线路一样, 此时一个电话呼叫和一个因特网连接能够同时共享一条 DSL 链路.

在 DSL 因特网接入中, 多个用户可以共享同一根电话线, 但是 DSL 通过频分复用等技术将不同用户的数据信号分隔在不同的频段上, 这意味着在 DSL 中, 每个用户的调制解调器接收到的数据信号是特定于他们的设备和频段的, 与其他用户的数据信号是分隔的.

在家庭一端, 分配器能把到达家庭的数据信号和电话信号分隔开, 并将数据信号转发给 DSL 调制解调器; 在电话公司一端, 本地中心局中的 DSLAM 把数据信号和电话信号分隔开, 并将数据送往因特网.

DSL 宽带住宅接入在传输过程中分为高速下行信道和中速上行信道, 这种设计考虑了用户在使用互联网时的典型需求和数据流量分布的特点, 原因如下:

  1. 数据流量不对称: 在大多数互联网应用中, 用户通常需要更大的下载带宽, 用于浏览网页、观看视频、下载文件等. 因此, 高速下行信道可以提供更高的带宽, 以满足用户在这些活动中的需求. 相比之下, 上传带宽需求通常较低, 例如上传文件、发送电子邮件等, 因此可以使用中速上行信道满足.
  2. 提供更好的用户体验: 分配不同的带宽给下行和上行信道可以在高峰时段维持更好的服务质量. 在高峰时段, 许多用户可能在同时进行下载等活动, 因此提供高速下行信道可以确保他们获得稳定和高速的连接. 而上传带宽通常不会因为用户活动而受到太大影响.
  3. DSL 技术特点: DSL 技术的下行和上行传输性能可能不同. 在某些情况下, 下行信号可能能够在电话线上传输更远, 而上行信号可能受到一些限制. 通过分配更高的带宽给下行信道, 可以更好地利用这些特点, 提供更远的覆盖范围.
  4. 成本和资源分配: 在典型的家庭和个人用户中, 下载活动通常更为频繁和密集, 因此在资源分配上倾向于满足这种需求.
电缆因特网接入

电缆因特网接入利用有线电视公司现有的有线电视基础设施. 住宅从提供有线电视的公司获得电缆因特网接入.

如图所示:

  • 光缆将电缆头端连接到地区枢纽, 从这里使用的同轴电缆到达各个家庭.
  • 电缆因特网接入需要特殊的调制解调器, 这种调制解调器称为电缆调制解调器. 它作为外部设备, 通过一个以太网端口连接家庭 PC.
  • 在电缆头端, 电缆调制解调器端接系统 (Cable Modem Termination System, CMTS) 将来自许多家庭的模拟信号转换为数字信号.
  • 电缆调制解调器将网络划分为下行和上行两个信道, 如同 DSL, 下行信道分配的传输速率通常比上行信道的高.

电缆因特网接入的一个重要特征是共享广播媒体. 在电缆因特网接入中, 每个用户的电缆调制解调器会收到相同的广播传输. 这是因为电缆网络的架构类似于共享广播媒体, 其中数据被广播到所有连接到电缆的用户. 尽管每个用户的设备可以识别并提取属于自己的数据, 但实际上, 所有用户在同一时间收到的数据是相同的.

光纤到户

一种提供更高速率的新兴技术是光纤到户 (Fiber To The Home, FTTH). 光纤到户是指从本地中心局直接到家庭提供一条光线路径.

从本地中心局到家庭有几种光纤分布方案:

  • 最简单的光纤分布网络称为直接光纤, 即从本地中心局到每个用户设置一条光纤.
  • 更一般的方案是: 从本地中心局出来的每根光纤由多个家庭共享, 直到接近这些家庭的位置, 该光纤才分成每户一根. 这种分布方案有两种光纤分布体系结构:
    • 主动光纤网络 (Active Optical Network, AON).
    • 被动光纤网络 (Passive Optical Network, PON). 使用 PON 分布体系结构的光纤到户如图所示:
      • 每个家庭拥有一个光纤网络端接器 (Optical Network Terminator, ONT), 它由专门的光纤连接到邻近的光纤分配器 (Optical splitter). 该分配器把一些家庭集结到一根共享光纤, 最后连接到中心局的光纤线路端接器 (Optical Line Terminator, OLT).
      • OLT 提供光信号和电信号之间的转换, 经过本地电话公司路由器与因特网相连接.
      • 在家庭中, 用户将一台家用路由器与 ONT 相连, 并经过这台家庭路由器接入因特网.
      • 在 PON 体系结构中, 所有从 OLT 发送到分配器的分组在分配器处被复制, 并发送给所有的家庭, 即每个家庭的 ONT 接收到的数据是相同的. 当然, ONT 会使用一些技术来确保数据只被特定家庭或用户设备解码并使用, 以保护用户的隐私和数据安全.
拨号

使用传统电话线的拨号接入与 DSL 基于相同的模式: 家庭的调制解调器经过电话线连接到 ISP 的调制解调器. 与 DSL 和其他宽带接入网相比, 拨号接入的速度比较慢.

卫星

在无法提供 DSL、电缆和光纤到户的地方, 能够使用卫星链路将住宅与因特网相连.

企业 (和家庭) 接入: 以太网和 WiFi

局域网

局域网 (Local Area Network, LAN) 是指在相对较小的地理范围内, 连接多台计算机、设备和资源的网络.

在公司和大学校园中, 使用局域网将端系统连接到边缘路由器.

以太网

以太网 (Ethernet) 是一种有线局域网技术. 用于在有限的范围内连接设备.

如图所示:

  • 以太网用户使用双绞铜线与一台以太网交换机相连, 以太网交换机再与更大的因特网相连.
WiFi

WiFi 是一种基于 IEEE 802.11 技术的无线局域网技术, 它允许设备通过无线信号在有限的范围内进行通信和连接.

无线接入点 (Access Point, AP) 是一种设备, 它通过无线局域网技术 (如 WiFi) 创建一个无线网络来允许多台无线设备连接到局域网. 无线接入点作为一个中继站, 将无线设备的数据传输转发到有线网络 (如以太网) 中. 无线接入点通常与有线网络连接, 使得无线设备可以通过无线网络连接到互联网或局域网.

尽管以太网和 WiFi 接入网络最初部署在企业中, 但它们现在也是家庭网络的常见组件. 许多家庭将宽带住宅接入 (即 DSL 因特网接入和电缆因特网接入) 与这些廉价的无线局域网技术相结合, 以创建强大的家庭网络.

下图是一个典型的家庭网络:

这个家庭网络组成如下:

  • 一台笔记本 (无线 PC)、多台联网家用电器以及一台有线 PC.
  • 基站 (无线接入点), 它用来和家庭中的无线 PC 和其他无线设备进行通信.
  • 家庭路由器 (不同于网络核心中的路由器), 用于将无线接入点 (现在的家庭路由器通常集成了无线接入点的功能) 和其他有线设备连接到互联网.

广域无线接入: 3G 、LET 4G 和 5G

iPhone 和 Android 设备采用与蜂窝电话相同的无线基础设施, 通过蜂窝网络提供商操作的基站来发送/接收分组.

4G 代表第四代移动通信技术, 是一种高速移动网络技术, 用于提供更快的数据传输速度和更稳定的连接, 以满足日益增长的移动通信需求. 4G 技术主要基于 LTE (Long-Term Evolution) 技术标准, 它在数据传输速率、延迟和网络容量方面显著提升了性能.

物理媒介

在计算机网络中, 每个比特从一个端系统出发, 经过一系列 “发射器 - 接收器” (路由器) 对, 到达另一个端系统. 对于每个 “发射器 - 接收器” 对, 通过一种物理媒介 (physical medium) 来传输电磁波或光脉冲来发送比特.

物理媒介可以具有多种形状和形式, 并且对于沿途的每一个 “发射器 - 接收器” 对而言不必具有相同的类型.

物理媒介分为两种类型:

  1. 导引型媒体 (guided media): 电波沿着固体媒介前行.
    • 如: 光缆、双绞铜线或同轴电缆.
  2. 非导引型媒体 (unguided media): 电波在空气或外层空间中传播.
    • 如: 无线局域网、数字卫星频道.

物理链路的实际成本与其他网络成本相比相对较小.

双绞铜线

最便宜且最常用的导引型传输媒介是双绞铜线, 它一直用于电话网.

双绞铜线由两根绝缘的铜线组成, 两者以规则的螺旋状排列着. 通常一根电缆由许多双绞线捆扎在一起形成. 其中无屏蔽双绞线 (Unshielded Twisted Pair, UTP) 常用在建筑内的计算机网络 (局域网) 中.

现在, 双绞线已经成为高速局域网联网的主导性解决方案.

双绞线也经常用于住宅因特网接入. 如数字用户线技术通过双绞线使住宅用户能够接入因特网.

同轴电缆

与双绞线类似, 同轴电缆由两个铜导体组成, 但是这两个导体是同心的而不是并行的.

借助这种结构以及特殊的绝缘体和保护层, 同轴电缆能够达到较高的数据传输速率.

同轴电缆能被用作导引型共享媒体. 当多个端系统直接与同轴电缆相连时, 每个端系统都能接收到由其他端系统发送的内容.

同轴电缆也经常用于住宅因特网接入. 如电缆因特网接入通过同轴电缆使住宅用户能够接入因特网.

光纤

光纤是一种细而柔软的、能够导引光脉冲的媒体, 每个脉冲表示一个比特.

光纤能够支持极高的比特速率而不受电磁干扰, 光纤信号衰减极低且不易被窃听, 这些特征使光纤成为长途导引型传输媒介.

光纤常被用于长途电话网络, 也被广泛用于因特网的主干. 由于光设备 (如发射器、接收器和交换机) 的成本高, 所以在短途运输中不使用光纤 (如局域网、家庭接入网).

陆地无线电信道

无线电信道承载电磁频谱中的信号. 它不需要安装物理线路, 并具有穿透墙壁、提供与移动用户的连接以及长距离承载信号的能力. 这一特性极大地依赖于传播环境和信号传输的距离.

陆地无线电信道主要分为三类:

  1. 运行在很短的距离 (如 1 米或 2米).
    • 如: 无线头戴式耳机、无线键盘.
  2. 运行在局域, 通常跨越数十到几百米的距离.
    • 如: 无线局域网技术使用的局域无线电信道.
  3. 运行在广域, 通常跨越数万米的距离.
    • 如: 蜂窝接入技术使用的广域无线电信道.

卫星无线电信道

一颗通信卫星通常连接两个或多个地面微波发射器/接收器 (也称为地面站). 卫星在特定的频率上接收地面站发送的信号, 然后通过一个叫做 “中继器” 的设备来重新生成信号, 最后以另一个频率将信号再次发送出去.

通信中常使用两种卫星: 同步卫星和近地轨道卫星.

卫星链路通常用于那些无法使用 DSL 或电缆因特网接入的区域.

网络核心

分组交换

在各种网络应用中, 端系统彼此交换报文.

  • 报文可以执行控制功能, 也可以包含数据.

为了从端系统向目的端系统发送一个报文, 源将较长的报文划分为较小的数据块, 称之为分组. 在源和目的地之间, 每个分组都通过通信链路和分组交换机传送.

分组以等于该链路最大传输速率的速度通过通信链路. 因此, 如果某源端系统或分组交换机通过一条链路发送一个 L 比特的分组, 假设链路的传输速率为 R 比特/秒, 则传输该分组所需的时间为 LR\frac{L}{R}RL​.

存储转发传输

多数分组交换机在链路的输入端使用存储转发传输机制. 存储转发传输是指在交换机能够开始向输出链路传输该分组的第一个比特之前, 必须接收到整个分组.
