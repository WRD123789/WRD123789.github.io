  1. 快捷键
  • “Tab”: 尝试根据你到目前为止所写的内容自动完成后面的填充.

文本编辑器

打开指定路径的文件.

命令 解释
Esc -> “:q” 关闭 Vim, 不保存文件
Esc -> “:wq” 关闭 Vim 并保存文件
Esc -> “:w” 保存文件
Esc -> “:q!” 强制关闭 Vim, 不保存文件
Esc -> “:wq!” 强制关闭 Vim, 并保存文件
Esc 只读模式
i 编辑模式, 可以编辑文件
Esc -> “x” 在只读模式下删除一个字符
Esc -> “dd” 在只读模式下删除一行
Esc -> “u” 在只读模式下撤回上一次删除命令

-size 指定文件大小 (单位为 k)
-empty 只搜索空的文件或空目录
-ctime 指定搜索创建在 n 小时前的文件
