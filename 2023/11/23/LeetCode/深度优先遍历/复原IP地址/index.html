LeetCode 93. 复原 IP 地址

题目链接

https://leetcode.cn/problems/restore-ip-addresses/description/

题目描述

有效 IP 地址 正好由四个整数 (每个整数位于 0 到 255 之间组成, 且不能含有前导 0), 整数之间用  ‘.’ 分隔.

  • 例如: “0.1.2.201” 和 “192.168.1.1” 是 有效 IP 地址, 但是 “0.011.255.245”、“192.168.1.312” 和 “192.168@1.1” 是 无效 IP 地址.

给定一个只包含数字的字符串 s , 用以表示一个 IP 地址, 返回所有可能的有效 IP 地址, 这些地址可以通过在 s 中插入 ‘.’ 来形成. 你 不能 重新排序或删除 s 中的任何数字. 你可以按 任何 顺序返回答案.

题解

算法过程

  • 深度优先遍历
  • 用递归来搜索答案, restNumbers 表示剩余的数字, needNumber 表示还需要多少个数字 (该题中总共需要 4 个数字), currentIpdAddress 表示当前已经拼接好的 IP 地址.
  • 每次搜索 restNumbers 的 0-2 位, 因为每个整数最大为 3 位.
  • 根据 needNumber 来决定什么时候结束搜索
    • 如果 needNumber < 0, 说明找到 4 个整数后 restNumbers 中还有剩余, 此时没有搜索下去的必要.
    • 如果 needNumber = 0 且 restNumbers 中没有剩余, 那么说明已经找到了一种 IP 地址, 在 Solution 类的 allIpAddresses 中记录结果即可.

时间复杂度分析

深搜第 k 层的节点数最多为 3k3^k3k, 由于限制了搜索深度, 最多只能搜索 4 层, 所以时间复杂度为 3+31+32+33=3×(1−33)1−3=O(1)3 + 3^1 + 3^2 + 3^3 = \frac{3 \times (1 - 3^3)}{1 - 3} = O(1)3+31+32+33=1−33×(1−33)​=O(1).
