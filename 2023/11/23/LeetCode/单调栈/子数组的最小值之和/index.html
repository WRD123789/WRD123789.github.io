LeetCode907. 子数组的最小值之和

题目描述

给定一个整数数组 arrarrarr, 找到 min(b)min(b)min(b) 的总和, 其中 bbb 的范围为 arrarrarr 的每个（连续）子数组.

由于答案可能很大, 因此返回答案模 109+710^9 + 7109+7.

算法过程

  • 单调栈
  • 最小值是一段连续的数字中的最小值, 所以每个最小值都有一定的辐射范围. 假设给定数组 A = [3,1,2,4,1]A \ = \ [3, 1, 2, 4, 1]A = [3,1,2,4,1], 在一段连续数字 3、1、2、4、13、1、2、4、13、1、2、4、1 中, 只要其中一段数字包含 111, 那么这段数字的最小值肯定为 111, 例如 [3,1,2,4,1]、[3,1,2,4]、[3,1,2]、[1,2][3, 1, 2, 4, 1]、[3, 1, 2, 4]、[3, 1, 2]、[1, 2][3,1,2,4,1]、[3,1,2,4]、[3,1,2]、[1,2] 等最小值都为 111, 我们把这叫做 111 的辐射范围. 显然数字 kkk 对最终答案的贡献值等于 k⋅nk \cdot nk⋅n, 其中 nnn 为辐射范围内能够产生的包含数字 kkk 的连续子数组的数量.
  • 那么如何求出 nnn 呢: 每个辐射范围都有一个左边界 leftleftleft 和右边界 rightrightright, 那么子数组的边界应该在 [left,i][left, i][left,i] 中选, 右边界应该在 [i,right][i, right][i,right] 中选.
  • 如何求取辐射范围: 左边碰到第一个 iii 满足 arr[i]≤karr[i] \leq karr[i]≤k, 则 iii 为 kkk 的辐射范围的左边界; 同理, 右边碰到第一个 iii 满足 arr[i]<karr[i] < karr[i]<k, 则 iii 为 kkk 的辐射范围的右边界, 此时可以用单调栈来解决.
  • 单调栈中存储下标, 以 arrarrarr 递增存储:
    • 如果 arr[i]<s.top()arr[i] < s.top()arr[i]<s.top(), 说明当前 iii 为栈顶的右边界, 而单调栈是以 arrarrarr 递增存储, 所以栈内离栈顶最近的元素为栈顶的左边界, 此时栈顶的辐射范围已经确定, 出栈, 直到栈顶小于 arr[i]arr[i]arr[i], 此时将 iii 加入栈即可.
    • 如果 arr[i]≥s.top()arr[i] \geq s.top()arr[i]≥s.top(), 直接将 iii 加入栈即可.

时间复杂度分析

因为每个下标最多进栈一次, 出栈一次, 所以时间复杂度为 O(N)O(N)O(N).
