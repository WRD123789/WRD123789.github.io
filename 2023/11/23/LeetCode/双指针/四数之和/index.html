LeetCode 18. 四数之和

题目链接

https://leetcode.cn/problems/4sum/

题目描述

给你一个由 n 个整数组成的数组 nums , 和一个目标值 target , 请你找出并返回满足下述全部条件且不重复的四元组 [nums[a],nums[b],nums[c],nums[d]][nums[a], nums[b], nums[c], nums[d]][nums[a],nums[b],nums[c],nums[d]] (若两个四元组元素一一对应, 则认为两个四元组重复):
- 0 ≤\leq≤ a, b, c, d < n
- a、b、c 和 d 互不相同
- nums[a] + nums[b] + nums[c] + nums[d] = target

注意：你可以按 任意顺序 返回答案.

算法流程

  • 排序, 双指针
  • 对数组按升序排序
  • 遍历排序后的数组 nums (在三数之和的基础上加一层循环即可):
    • 对于重复元素直接跳过, 避免出现重复解.
    • 设置左指针 left_p = i + 1, 右指针 right_p = n -1, while 循环, 直到两指针相遇:
      • 如果 nums[i]nums[i]nums[i] + nums[j]nums[j]nums[j] + nums[left_p]nums[left\_p]nums[left_p] + nums[right_p]nums[right\_p]nums[right_p] = target, 则对左右指针分别判断是否有重复项, 有则剔除.
      • 如果 nums[i]nums[i]nums[i] + nums[j]nums[j]nums[j] + nums[left_p]nums[left\_p]nums[left_p] + nums[right_p]nums[right\_p]nums[right_p] < target, 说明和比较小, 移动左指针来增加和.
      • 如果 nums[i]nums[i]nums[i] + nums[j]nums[j]nums[j] + nums[left_p]nums[left\_p]nums[left_p] + nums[right_p]nums[right\_p]nums[right_p] > target, 说明和比较大, 移动右指针来减小和.

时间复杂度分析

  • 对数列排序所需的时间复杂度为 O(Nlog⁡2NN\log_2^{N}Nlog2N​), 遍历数组所需时间复杂度为 O(N2N^{2}N2), 双指针遍历所需时间复杂度为 O(N), 则总的时间复杂度为 O(Nlog⁡2NN\log_2^{N}Nlog2N​) + O(N3N^{3}N3) = O(N3N^{3}N3).
