字符串

当比较两个字符串大小的时候, 不能直接用运算符 <, >, ==, 而应当使用 str1.compareTo(str2)

compareTo 方法返回值:

  • 如果 str1 < str2, 返回负数.
  • 如果 str1 == str2, 返回0.
  • 如果 str1 > str2, 返回正数.
