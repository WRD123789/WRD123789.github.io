编译

终端执行:

  • 要运行 xxx.java 文件, 需要两步:

    • javac xxx.java
    • java xxx

  • 其中第一步会产生一个 xxx.class 文件

  • 运行 .java 文件其实如下图所示:

  • 将 xxx.java 文件给编译器, 生成其类文件, 即 xxx.class 文件, 然后再将该类文件给解释器, 解释器执行相应的事件.

生成类文件原因:

  • .class 文件经过变量的类型检查, 将经过类型检查的 .class 文件交给客户更安全, 可以减少错误.
  • .class 文件更容易被机器执行, 所以 .class 文件执行更快.
  • 可以保护知识产权, 使别人无法知道你的代码实现 (因为交给别人的是 .class 文件而不是 .java 文件).
