  • 由于泛型的工作方式, 我们无法创建基本类型的集合.
    • 例如无法创建 Set<int> .
  • 为了更方便使用这些集合, Java 进行了一些自动类型转换.

上文中的 List、Set 和 Map 都是接口: 它们定义这些相应类型的工作方式, 但不提供具体的实现代码. 这使得这些类型的用户可以在不同的情况下选择不同的实现.

  • 例如:
    • ArrayList 和 LinkedList 是 List 接口的两种实现. 它们都提供了 List 要求的所有方法, 并且这些方法必须按照 List 的文档所描述的那样工作; HashSet 和 HashMap 分别是 Set 接口和 Map 接口的实现.
    • 还可以结合上文中的 List.of 进行初始化
