枚举

当在 Java 中需要一个固定数量的常量集合时, 可以使用枚举类型 (enum), 枚举类型是 Java 语言中一种特殊的类, 它被用来表示一个固定的值集合, 这些值在整个程序中是不可变的

枚举常量默认是 public static final 修饰的, 所以可以通过类名直接访问它们:

相比与用数字或字符串来表示一组有限的值, 用枚举类型更加安全, 它能够捕捉类型不匹配之类的错误
